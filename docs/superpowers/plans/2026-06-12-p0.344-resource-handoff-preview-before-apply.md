# P0.344 Resource Handoff Preview Before Apply Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Replace the provider-selection missing operator action from `make final-apply-resources` to `make final-resource-apply-preview`, while leaving the command catalog unchanged.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, ruff.
