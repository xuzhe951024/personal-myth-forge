Change `_is_drop_candidate_source_prefix()` so it accepts `bare_root` and only
returns true for the current mobile preflight evidence prefix or for roots in
`FINAL_SHOWCASE_DUPLICATE_DEVICE_ROOTS`. The final implementation also first

```python
def _is_drop_candidate_source_prefix(action: str, bare_root: str) -> bool:
    if not _is_source_prefixed_action(action):
        return False
    if action.startswith(
        "final_rehearsal_local: mobile_deploy_preflight_evidence: "
    ):
        return True
    return bare_root in FINAL_SHOWCASE_DUPLICATE_DEVICE_ROOTS
```
