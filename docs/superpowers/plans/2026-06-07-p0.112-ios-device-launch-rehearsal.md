# P0.112 iOS Device Launch Rehearsal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `make ios-device-launch-rehearsal`, a safe one-command local rehearsal that refreshes final handoff reports and writes an ignored iOS device launch rehearsal summary.

**Architecture:** Add a backend read-only report builder that summarizes saved final configured preflight, final handoff index, and iOS device launch certificate JSON files. Add a shell wrapper that runs the existing safe report-writing commands, accepts exit `0` and `2`, then calls the new CLI to write the rehearsal summary.

**Tech Stack:** Python 3.10, pytest, POSIX shell, Makefile, Swift source gate checks.
