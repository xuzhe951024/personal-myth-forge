> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a structured final showcase readiness `next_action` and render it first on the iPhone Final Launch readiness panel.

**Architecture:** Derive `next_action` from the existing `first_blocker`/first non-ready capability in `final_showcase_readiness.py`, sanitize it with the existing report sanitizer, and expose it through Swift models. The mobile summary builder renders the structured row before the existing capability/action rows, while source and visual gates prevent regression.

**Tech Stack:** Python 3.10, pytest, FastAPI/Pydantic-adjacent dict reports, Swift Package Manager, Swift Codable models, static HTML visual regression.
