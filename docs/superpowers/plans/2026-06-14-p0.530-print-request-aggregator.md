> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Reuse the shared print action preference helper so aggregators do not carry stale guarded quote actions when a request-preflight action is present. Let aggregator action builders prefer concrete child `operator_action` values before falling back to generic validation commands.

**Tech Stack:** Python, FastAPI backend support modules, pytest, Make safe wrappers.
