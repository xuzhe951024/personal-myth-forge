```python
CONFIGURED_PRINT_QUOTE_ACTION = (
    "PMF_ALLOW_PRINT_PROVIDER_CALLS=1 make print-quote-configured"
)
CONFIGURED_PRINT_QUOTE_VALIDATED_ACTION = (
    f"{CONFIGURED_PRINT_QUOTE_ACTION}; rerun make print-fulfillment-readiness"
)
PRINT_FULFILLMENT_READINESS_ACTION = "make print-fulfillment-readiness"
```

- [x] **Step 2: Implement the minimal filter**

Add:

```python
def _prefer_guarded_print_quote_action(actions: list[str]) -> list[str]:
    has_guarded_print_quote = any(
        _operator_action_root(action) == CONFIGURED_PRINT_QUOTE_VALIDATED_ACTION
        for action in actions
    )
    if not has_guarded_print_quote:
        return actions
    return [
        action
        for action in actions
        if _operator_action_root(action) != PRINT_FULFILLMENT_READINESS_ACTION
    ]
```

Call it after `_prefer_apply_preview_before_apply(_dedupe(actions))`.
