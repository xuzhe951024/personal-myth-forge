Add a helper:

```python
def _prefer_apply_preview_before_apply(actions: list[str]) -> list[str]:
    if FINAL_RESOURCE_APPLY_PREVIEW_COMMAND not in {
        _operator_action_command(action) for action in actions
    }:
        return actions
    return [
        action
        for action in actions
        if _operator_action_command(action) != FINAL_RESOURCE_APPLY_COMMAND
    ]
```
