# P0.338 iOS Specific Deploy Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Drop generic iOS deploy config actions from final handoff lists when the concrete project-local deploy writer command is already present.

**Architecture:** Extend the shared operator action preference helper so all report builders using it inherit the same writer-over-generic behavior. Keep low-level diagnostics intact and only filter top-level action list entries.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, ruff.
