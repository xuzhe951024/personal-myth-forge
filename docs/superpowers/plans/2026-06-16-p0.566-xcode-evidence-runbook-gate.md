# P0.566 Xcode Evidence Runbook Gate Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make resource handoff and iOS deploy runbook honor saved ready Xcode build evidence.

**Architecture:** Add local report helpers in the two affected report builders. The helpers read `services/backend/.local/mobile-xcode-build-evidence.json`, validate the report kind/status/classification/exit code/safety fields, and use that readiness to switch only the Xcode gate from manual to ready.

**Tech Stack:** Python, pytest, existing JSON `.local` reports.
