# P0.559 Local Demo Drop Weak Apply Preview Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove redundant bare `make final-resource-apply-preview` top-level actions from local final demo reports when the complete provider handoff chain is already present.

**Architecture:** Reuse the existing complete provider chain as the canonical resource/provider action. Add one focused helper in `final_demo_launch.py`, and extend the existing weak provider root set in `final_local_report_refresh.py`.

**Tech Stack:** Python 3.10, FastAPI backend report builders, pytest, Make wrappers.
