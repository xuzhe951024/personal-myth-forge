# P0.63 Final Demo Launch Profile Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a source-tested final demo launch profile report that tells the operator how to move from resource handoff to local/configured iPhone showcase acceptance without leaking secrets or mutating global state.

**Architecture:** Introduce `myth_forge_api.final_demo_launch` as a thin orchestration report over existing `resource_handoff` data. Wire a new CLI command and Make target, update resource handoff Treatstock wording, and extend resource template acceptance so the final launch surface is tested.

**Tech Stack:** Python 3.10, pytest, argparse CLI, Make, SwiftPM source checks, static HTML/PNG evidence.
