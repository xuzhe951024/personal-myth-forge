# P0.8 iOS App Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a checked-in Xcode iOS app shell for the existing SwiftUI scaffold and verify it locally without changing global Xcode settings.

**Architecture:** Keep the SwiftPM core as the local testable contract surface. Add an Xcode project shell that references the same Swift files, plus a SwiftPM executable inspector that validates the project/plist/docs as data because `xcodebuild` cannot run under Command Line Tools.

**Tech Stack:** Swift 6.3, Swift Package Manager executable checks, Xcode project file, iOS `Info.plist`, SwiftUI source scaffold, FastAPI backend verification.

---

## Environment Constraint

Do not run `xcode-select` or change the active developer directory. Local
deployment builds require a full Xcode installation and are documented as a
follow-up manual step.

## File Structure

- Create `apps/mobile/ios/PersonalMythForge.xcodeproj/project.pbxproj`: minimal iOS app target shell.
- Create `apps/mobile/ios/App/Info.plist`: bundle metadata, permissions, backend base URL key.
- Create `apps/mobile/ios/App/AppConfiguration.swift`: bundle-backed backend URL lookup.
- Modify `apps/mobile/ios/Package.swift`: add `PersonalMythForgeMobileProjectChecks` executable target.
- Create `apps/mobile/ios/Sources/PersonalMythForgeMobileProjectChecks/main.swift`: project/plist/docs inspector.
- Modify `apps/mobile/README.md`: Xcode open guidance and SwiftPM project-check command.
- Modify `README.md`: mention P0.8 app shell verification.
- Modify `docs/iteration-roadmap.md`: mark P0.8 app shell bridge deliverable.

- [x] **Step 2: Add app configuration source**

Create `AppConfiguration.swift`:

```swift
import Foundation

enum AppConfiguration {
    static var backendBaseURL: URL {
        if let rawValue = Bundle.main.object(forInfoDictionaryKey: "PMFBackendBaseURL") as? String,
           let url = URL(string: rawValue) {
            return url
        }
        return URL(string: "http://127.0.0.1:8080")!
    }
}
```
