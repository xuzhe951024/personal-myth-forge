
Create `NPCAgentTickSummary` with `status`, `title`, `detail`, `runtimeLabel`, `tickLabel`, `decisionLabel`, `rows`, and `privacyNotes`.

- [ ] **Step 2: Implement `NPCAgentTickSummaryBuilder.build`**

Use this signature:

```swift
public static func build(
    session: MythSession?,
    latestTick: NPCAgentTick?,
    tickHistoryCount: Int,
    isAdvancingTick: Bool,
    isRunningAutonomy: Bool,
    errorMessage: String?
) -> NPCAgentTickSummary
```

Behavior:

- error text returns `.needsAttention`
- no session returns `.waiting`
- `isRunningAutonomy` returns `.running` with `3-step autonomy run`
- `isAdvancingTick` returns `.running` with `single NPC tick`
- no latest tick returns `.ready` using session traces
- latest tick returns `.ready` using tick runtime and world resolution counts
