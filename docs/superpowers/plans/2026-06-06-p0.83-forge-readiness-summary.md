### Task 2: Core Builder

**Files:**
- Create: `apps/mobile/ios/Sources/PersonalMythForgeMobileCore/ForgeReadinessSummary.swift`

- [ ] **Step 1: Implement public API**

Create:

```swift
public enum ForgeReadinessSummaryStatus: String, Codable, Equatable, Sendable {
    case waiting
    case ready
    case needsAttention
}
```

Create `ForgeReadinessSummary` with `status`, `title`, `detail`, `routeLabel`, `rows`, `privacyNotes`, and `canForge`.

- [ ] **Step 2: Implement builder**

Rules:

- provider readiness error -> `.needsAttention`, `canForge = false`
- capture not ready -> `.waiting`, `canForge = false`
- context not ready -> `.waiting`, `canForge = false`
- missing 3D or NPC provider setup -> `.needsAttention`, `canForge = false`
- capture/context ready with local demo providers -> `.ready`, `routeLabel = "local_demo"`, `canForge = true`
- real providers ready -> `.ready`, `routeLabel = "configured"`, `canForge = true`
