# P0.17 NPC Agent Runtime Trace Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use
> superpowers:subagent-driven-development or superpowers:executing-plans to
> implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for
> tracking.

**Goal:** Add a backward-compatible NPC agent trace contract so the backend can
expose per-NPC beliefs, intentions, proposed actions, rationales, and confidence
while the mobile app can decode and show those traces.

**Architecture:** Keep existing `npc_reactions` and `world_resolution` intact.
Add trace models to the backend and Swift core, generate deterministic local
traces, update OpenAI structured output handling, and render compact trace rows
in the iOS source.

**Tech Stack:** FastAPI, Pydantic, pytest, Swift 6, SwiftPM, SwiftUI.
