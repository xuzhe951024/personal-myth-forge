# Final Local Self-Rerun Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove redundant `; rerun <same command>` suffixes from `final-local-report-refresh` operator actions.

**Architecture:** Keep the existing operator action aggregation pipeline. Add one command-root comparison inside `_action_command` so identical command/validation pairs return the command once, while differing validation commands still produce validation-aware handoff strings.

**Tech Stack:** Python 3.10, pytest, existing backend report builders.
