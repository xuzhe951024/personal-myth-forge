# P0.421 Mobile Evidence Device Bundles Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add standardized device action bundles to bottom-level mobile deploy preflight and mobile Xcode build evidence reports.

**Architecture:** Each evidence report already computes checks, blockers, and next actions. Add local helper functions in each module to convert those existing values into a sanitized `device_action_bundle`, without changing the scripts, command execution, readiness classification, or higher-level report behavior.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, existing Make/CLI report targets.
