- [ ] **Step 3: Implement formatting**

Add a helper:

```python
def _report_blocker_detail(blocker: dict[str, Any]) -> str:
    next_action = blocker.get("next_action")
    if isinstance(next_action, dict):
        command = str(next_action.get("command", "")).strip()
        validation = str(next_action.get("validation_command", "")).strip()
        detail = str(next_action.get("detail", "")).strip()
        parts = [f"Next action: {command}" if command else ""]
        if validation:
            parts.append(f"rerun {validation}")
        if detail:
            parts.append(detail)
        formatted = " | ".join(part for part in parts if part)
        if formatted:
            return formatted
    return str(blocker.get("detail", blocker.get("status", "blocked")))
```

Use it inside `_simple_report_capability()`.
