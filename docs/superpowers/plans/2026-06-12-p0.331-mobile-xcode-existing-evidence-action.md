# Existing Xcode Evidence Action Normalization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ensure `final-local-report-refresh` upgrades existing local mobile
Xcode evidence reports with `first_blocker` and `next_action` without running
Xcode.

**Architecture:** Keep preserving existing evidence, but pass it through a
schema-normalization helper before returning. The helper only adds missing
action fields and leaves existing status, classification, checks, safety, and
captured output intact.

**Tech Stack:** Python 3.10, pytest, FastAPI backend CLI, ignored `.local`
reports.
