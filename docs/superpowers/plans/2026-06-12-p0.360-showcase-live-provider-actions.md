### Task 2: Implement Capability Action Formatting

**Files:**
- Modify: `services/backend/src/myth_forge_api/final_showcase_readiness.py`

- [ ] **Step 1: Add a helper for capability-derived actions**

Add:

```python
def _capability_operator_action(row: dict[str, Any]) -> str:
    command = str(row.get("command", "")).strip()
    validation_command = str(row.get("validation_command", "")).strip()
    if command and validation_command:
        return f"{command}; rerun {validation_command}"
    return command
```

- [ ] **Step 2: Use the helper in `_operator_actions`**

Replace the capability action extension so it calls `_capability_operator_action`
for required non-ready rows and skips empty strings.

- [ ] **Step 3: Prefer validation-aware actions in dedupe**

Add a small dedupe pass that detects combined actions of the form
`<command>; rerun <validation>` and removes bare actions whose root equals the
combined command or validation command.
