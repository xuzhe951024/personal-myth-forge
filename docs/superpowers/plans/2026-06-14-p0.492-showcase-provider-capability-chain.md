### Task 2: Compose Provider Handoff Into Capability Metadata

**Files:**
- Modify: `services/backend/src/myth_forge_api/final_showcase_readiness.py`

- [x] **Step 1: Add validation composition helper**

Add:

```python
def _operator_action_with_validation(command: str, validation_command: str) -> str:
    if not command:
        return ""
    if not validation_command:
        return command
    validation_suffix = f"; rerun {validation_command}"
    if validation_suffix in command:
        return command
    return f"{command}{validation_suffix}"
```

- [x] **Step 2: Use source blocker validation command**

In `_report_next_action_metadata`, read:

```python
source_validation_command = str(
    next_action.get("source_blocker_validation_command", "")

```

If both `command` and `source_validation_command` are present, set:

```python
command = _operator_action_with_validation(command, source_validation_command)
```

Return the existing `validation_command` unchanged.

- [x] **Step 3: Avoid duplicate validation suffixes**

Update `_next_action_operator_action()` and `_capability_operator_action()` to
call `_operator_action_with_validation(command, validation_command)` instead of
hand-building `f"{command}; rerun {validation_command}"`.
