# P0.592 iOS Rehearsal Runbook Convergence Plan

## Goal

Prevent `ios-device-launch-rehearsal` from preserving a stale runbook snapshot that makes `final_handoff_index` appear blocked by its own device rehearsal loop.

## Steps

1. Inspect current git status, design docs, roadmap, and `.local` readiness reports.
2. Reproduce the stale chain with safe local wrapper commands.
3. Add RED tests to the iOS device rehearsal wrapper script harness.
4. Update `write_ios_device_launch_rehearsal.sh` to refresh `ios-deploy-runbook-local` after `final-rehearsal-local`.
5. Run focused GREEN tests, then relevant broader validation.
6. Refresh safe local evidence and verify the first blocker moved away from `final_handoff_index`.

## Safety

This phase does not run live provider calls, write secrets, start the device backend, perform Xcode signing, or submit print orders.
