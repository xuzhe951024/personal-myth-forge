# P0.428 Final Local Refresh Mobile Source Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a read-only final-local-report-refresh source to final demo launch and display it on iPhone Final Launch.

**Architecture:** Backend `final_demo_launch.py` reads the ignored `.local/final-local-report-refresh.json` snapshot without running commands and embeds it as `final_local_report_refresh`. Swift decodes compact refresh models and renders sanitized `localRefreshRows` in `FinalLaunchMobileSummary`.

**Tech Stack:** Python 3.10, FastAPI report builders, pytest, Swift 6, SwiftPM contract tests.
