- [x] **Step 1: Normalize rerun chains at the end of `normalize_operator_action()`**

Add a helper that accepts the final normalized string and returns a canonical
command chain:

```python
def _dedupe_rerun_chain(action: str) -> str:
    command_part, detail_suffix = _split_detail_suffix(action)
    parts = command_part.split("; rerun ")
    if len(parts) == 1:
        return f"{command_part.rstrip(';').strip()}{detail_suffix}"

    root = parts[0].rstrip(";").strip()
    validations: list[str] = []
    seen: set[str] = set()
    for raw_validation in parts[1:]:
        validation = raw_validation.rstrip(";").strip()
        if not validation or validation in seen:
            continue
        seen.add(validation)
        validations.append(validation)
    if not validations:
        return f"{root}{detail_suffix}"
    return f"{root}; rerun {'; rerun '.join(validations)}{detail_suffix}"
```

Then wrap each return from `normalize_operator_action()` with
`_dedupe_rerun_chain(...)`.
