# P0.390 Provider Handoff Resource Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the provider handoff report guide operators through ignored final resources, preview, apply, and validation commands.

**Architecture:** Keep the change inside `myth_forge_api.provider_handoff`; it already owns the provider report. Tests stay in `tests/test_cli.py` because the public behavior is the `provider-handoff` CLI JSON contract.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, existing `Settings` and provider readiness models.
