# iOS Certificate Handoff Priority Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Carry provider and Treatstock print handoff actions from `final-handoff-index` into `ios-device-launch-certificate`.

**Architecture:** Add optional `operator_actions` to device gates. The `final_handoff_index` gate receives only provider/print handoff actions extracted from the computed final handoff index, and `_operator_actions` prioritizes them after the first gate action.

**Tech Stack:** Python 3.10, pytest, existing backend report builders.

```python
def _final_handoff_priority_actions(report: dict[str, Any]) -> list[str]:
    actions = report.get("operator_actions")
    if not isinstance(actions, list):
        return []
    selected: list[str] = []
    for action in actions:
        if not isinstance(action, str):
            continue
        stripped = action.strip()
        if _is_provider_handoff_action(stripped) or _is_print_handoff_action(stripped):
            selected.append(stripped)
    return selected
```
