**Architecture:** The shared `myth_forge_api.operator_actions.add_mobile_deploy_validation_command()` helper already owns mobile validation suffixing. This phase makes that helper canonicalize repeated mobile preflight reruns while preserving detail suffixes and unrelated validation chains.
