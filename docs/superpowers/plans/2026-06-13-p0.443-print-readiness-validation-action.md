### Task 2: Chain Validation In Operator Actions

**Files:**
- Modify: `services/backend/src/myth_forge_api/print_fulfillment_readiness.py`

- [ ] **Step 1: Implement minimal chaining**

Change `_operator_actions()` so each non-ready command is returned as `<command>; rerun make print-fulfillment-readiness` and remove the unconditional standalone validation command.

```python
def _operator_actions(checks: list[dict[str, Any]]) -> list[str]:
    validation_command = "make print-fulfillment-readiness"
    actions = [
        f"{check['command']}; rerun {validation_command}"
        for check in checks
        if check.get("status") != "ready"
    ]
    if not actions:
        actions.append(validation_command)
    return _dedupe(actions)[:8]
```
