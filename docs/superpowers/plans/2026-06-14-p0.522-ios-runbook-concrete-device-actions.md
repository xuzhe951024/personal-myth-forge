# P0.522 iOS Runbook Concrete Device Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make iOS deploy runbook device action bundle commands include their mobile deploy preflight validation rerun.

**Architecture:** The runbook builds device slot actions in `ios_deploy_runbook._device_slot_action`. This phase applies the existing `add_mobile_deploy_validation_command()` helper at that boundary so all copied downstream reports receive validation-aware commands from the source.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, Makefile verification targets.
