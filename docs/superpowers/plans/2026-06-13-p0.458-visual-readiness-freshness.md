> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Block final showcase visual readiness when the saved visual regression report does not cover the current checked-in visual artifact inventory.

**Architecture:** `visual_regression_readiness.py` remains the single read-only adapter from `.local/visual-regression-local.json` to readiness status. It imports the current visual artifact inventory from `visual_regression.py`, compares saved artifact ids and pass count, and emits a normal blocker that upstream reports already understand.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, project-local ignored `.local` JSON reports.

- [ ] **Step 2: Check saved report coverage before readiness status**

Add a helper that returns a blocker when current artifact ids are missing or the saved passed count is lower than the current inventory size.

```python
def _inventory_blocker(
    *,
    summary: dict[str, int],
    artifacts: list[dict[str, Any]],
) -> dict[str, Any] | None:
    expected_ids = [artifact.id for artifact in DEFAULT_VISUAL_ARTIFACTS]
    saved_passed_ids = {
        str(artifact.get("id"))
        for artifact in artifacts
        if artifact.get("status") == "passed"
    }
    missing_ids = [
        artifact_id for artifact_id in expected_ids if artifact_id not in saved_passed_ids
    ]
    if missing_ids:
        return _blocker(
            classification="visual_regression_inventory_stale",
            detail=(
                "Saved visual regression report is missing current visual artifact "
                f"{missing_ids[0]}."
            ),
        )
    if summary["passed"] < len(expected_ids):
        return _blocker(
            classification="visual_regression_inventory_stale",
            detail=(
                "Saved visual regression report passed "
                f"{summary['passed']} artifacts but current inventory has {len(expected_ids)}."
            ),
        )
    return None
```

Call this helper in `_readiness_blocker()` after the empty pass count check and before failed artifact checks.
