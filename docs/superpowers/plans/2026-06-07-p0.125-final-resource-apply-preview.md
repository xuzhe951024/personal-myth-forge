# P0.125 Final Resource Apply Preview Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a sanitized dry-run report that previews what `make final-apply-resources` would write before the operator applies final API/key and iOS deploy resources.

**Architecture:** Build a new backend report from `final_resources_preflight` and `final_resource_requirements`, expose it through CLI/Make, embed it in Final Demo Launch, and render it in the iPhone Final Launch panel. The report is read-only and never shells out to the existing writers.

**Tech Stack:** Python 3.10, pytest, argparse CLI, SwiftPM, Swift Codable, existing Final Launch mobile summary/view patterns.
