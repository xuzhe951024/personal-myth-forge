# P0.124 Final Resource Requirements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a sanitized final resource requirements manifest that tells the operator which API keys, iOS deploy values, print values, and validation commands are required for final showcase readiness.

**Architecture:** Build a new backend report from `final_resources_preflight` so resource state has one source of truth while the new manifest adds operator metadata. Embed it in `final-demo-launch`, expose it through CLI/Make, decode it in Swift, and add read-only iPhone Final Launch rows.

**Tech Stack:** Python 3.10, pytest, argparse CLI, SwiftPM, Swift Codable, existing Final Launch mobile summary/view code.
