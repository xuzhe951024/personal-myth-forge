# P0.145 Final Resource Placeholder Validation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prevent example-only final resource values from being reported as configured or applied into backend/iOS local config.

**Architecture:** Extend the existing final resource preflight item builders so required iOS values can classify known placeholders separately from missing and loopback values. Keep apply preview downstream of preflight, and mirror the same guard in the shell apply script so direct script use cannot write placeholders.

**Tech Stack:** FastAPI backend Python reports, pytest, POSIX shell resource apply script.
