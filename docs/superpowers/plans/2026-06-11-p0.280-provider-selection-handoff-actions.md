# P0.280 Provider Selection Handoff Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace bare provider selector handoff actions with the existing safe final resource apply action.

**Architecture:** Keep provider selector values out of `final-resources.env`. Normalize stale provider selector actions in the shared operator action helper, and make `resource_handoff` emit the final apply action directly when provider selectors are not ready.

**Tech Stack:** Python, backend report builders, pytest, ruff.
