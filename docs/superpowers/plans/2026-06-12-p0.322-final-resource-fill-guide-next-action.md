# P0.322 Final Resource Fill Guide Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose a top-level `next_action` from final resource fill guide reports and show it in the iPhone final launch summary.

**Architecture:** Reuse the existing `first_blocker` object as the source of the actionable command. The backend report adds `next_action`, and the Swift model extends the existing first-blocker type with optional `source` so the same JSON shape can decode both fields.

**Tech Stack:** Python report builders, pytest, Swift Codable models, SwiftPM contract tests.
