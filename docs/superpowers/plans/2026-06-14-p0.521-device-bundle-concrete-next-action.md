# P0.521 Device Bundle Concrete Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Promote saved validation-aware device next actions into copied final handoff device bundle first actions.

**Architecture:** `final_showcase_readiness` already attaches `saved_next_action` when operator actions are more concrete than the raw device action. This phase updates downstream copy helpers in `final_demo_launch` and `final_local_report_refresh` so copied first actions display the saved command while preserving provenance fields.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, Makefile verification targets.
