# P0.323 Final Resource Apply Preview Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose and display a top-level `next_action` from final resource apply preview reports.

**Architecture:** Reuse the existing apply-preview `first_blocker` shape as the next-action contract. The backend adds `next_action`; Swift extends the first-blocker model with optional `source` and adds a summary row.

**Tech Stack:** Python report builders, pytest, Swift Codable models, SwiftPM contract tests.
