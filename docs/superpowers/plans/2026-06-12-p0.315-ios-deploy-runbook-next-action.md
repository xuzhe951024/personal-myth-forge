# P0.315 iOS Deploy Runbook Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add structured `first_blocker` and `next_action` fields to the iOS deploy runbook and expose them through the iOS final launch summary.

**Architecture:** The backend derives blocker/action fields from existing `input_slots` and `command_sequence`; Swift decodes optional fields and renders sanitized rows. Existing arrays stay unchanged for compatibility.

**Tech Stack:** Python/FastAPI report builder, pytest, Swift Package contract executable.
