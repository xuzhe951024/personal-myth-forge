# P0.221 Mobile Auto Backend URL Handoff Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Surface the `PMF_BACKEND_BASE_URL=auto` / `apply_time_auto` handoff that backend final resource reports already emit inside the iPhone Final Launch panel.

**Architecture:** Keep the backend resource reports as the source of truth. Extend the Swift Codable models to decode `resolution_mode` and `apply_note`, render those fields in existing Final Launch resource rows, and protect the handoff with Swift contract tests, backend source gates, and one static 390x844 visual regression artifact.

**Tech Stack:** Swift Codable and contract tests, FastAPI backend source acceptance tests, checked-in static visual regression HTML/PNG artifacts.
