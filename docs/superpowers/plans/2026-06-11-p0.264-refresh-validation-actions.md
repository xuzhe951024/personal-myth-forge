### Task 2: Preserve validation command during action formatting

**Files:**
- Modify: `services/backend/src/myth_forge_api/final_local_report_refresh.py`

- [x] **Step 1: Update `_action_command()`**

Replace the helper body with:

```python
def _action_command(action: dict[str, Any] | None) -> str:
    if not isinstance(action, dict):
        return ""
    command = str(action.get("command") or "").strip()
    validation_command = str(action.get("validation_command") or "").strip()
    if command and validation_command:
        return f"{command}; rerun {validation_command}"
    return command
```
