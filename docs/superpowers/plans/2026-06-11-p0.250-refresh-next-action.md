> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a sanitized top-level `next_action` to `final-local-report-refresh.json` by deriving it from the existing `first_blocker`.

**Architecture:** Keep the report generator as the single owner of refresh report shape. Compute `first_blocker` once, derive `next_action` from it, and keep all data passing through the existing report sanitizer.

**Tech Stack:** Python, FastAPI backend package, pytest, project Make targets.

- [ ] **Step 2: Add helper**

Add this helper near `_first_blocker`:

```python
def _next_action(first_blocker: dict[str, Any] | None) -> dict[str, Any] | None:
    if not isinstance(first_blocker, dict):
        return None
    return {
        "id": str(first_blocker.get("id", "")),
        "label": str(first_blocker.get("label", "")),
        "status": str(first_blocker.get("status", "")),
        "classification": str(first_blocker.get("classification", "")),
        "command": str(first_blocker.get("command", "")),
        "detail": str(first_blocker.get("detail", "")),
        "source": "first_blocker",
        "output": first_blocker.get("output"),
        "step_id": str(first_blocker.get("step_id", "")),
    }
```
