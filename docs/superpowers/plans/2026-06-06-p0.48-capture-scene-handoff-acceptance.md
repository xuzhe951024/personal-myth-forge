# P0.48 Capture Scene Handoff Acceptance Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a final acceptance gate proving the guided-scan upload path returns iOS-downloadable generated assets.

**Architecture:** Build a focused backend acceptance runner around FastAPI `TestClient`. It temporarily patches the app's capture store to a temp directory, runs the actual object capture and from-capture endpoints, downloads returned local generated assets, restores the patch, and emits a sanitized JSON report. Final acceptance consumes the runner as an in-process check.

**Tech Stack:** FastAPI TestClient, pytest, Pydantic response contracts, local capture store, deterministic local generated assets.
