### Task 2: Select The Auto Writer For Backend URL Device Actions

**Files:**
- Modify: `services/backend/src/myth_forge_api/mobile_deploy_preflight_evidence.py`

- [ ] **Step 1: Add helper**

Add:

```python
def _raw_action_for_check(
    *,
    check: dict[str, str],
    index: int,
    raw_actions: list[str],
) -> str | None:
    if (
        check.get("id") == "backend_base_url"
        and MOBILE_WRITE_DEPLOY_CONFIG_AUTO_ACTION in raw_actions
    ):
        return MOBILE_WRITE_DEPLOY_CONFIG_AUTO_ACTION
    if index < len(raw_actions):
        return raw_actions[index]
    return None
```
