# P0.212 Final Launch Certificate Preflight Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Surface the existing iOS device launch certificate inside final-demo-launch and mobile Device Preflight.

**Architecture:** Extend the backend final launch aggregator to embed `ios_device_launch_certificate` without recursive report construction by passing a base final demo launch report into the certificate builder. Add Swift Codable models and one required Device Preflight item that reads the certificate. Guard the wiring with backend pytest, Swift contract tests, ProjectChecks, and mobile final launch source acceptance.

**Tech Stack:** Python/FastAPI report builders, pytest, Swift Codable contract tests, SwiftPM ProjectChecks.
