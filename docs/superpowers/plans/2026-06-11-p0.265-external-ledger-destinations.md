**Goal:** Include resource destinations and rerun validation commands in `final-external-action-ledger.operator_actions`.

**Architecture:** Add a small formatting helper used only by `_operator_actions()` when a missing action requires user input and belongs to the `resource_inputs` group. The helper will derive the resource slot from the action id, use `Path(destination).name`, and append `; rerun <command>` when a command is present.

- [x] **Step 2: Add `_missing_user_input_action_text()`**

Add:

```python
def _missing_user_input_action_text(action: dict[str, Any]) -> str:
    slot = str(action["id"]).removeprefix("provide_")
    destination = Path(str(action.get("destination", ""))).name
    command = str(action.get("command", "")).strip()
    parts = [f"provide {slot}"]
    if destination:
        parts[0] = f"{parts[0]} in {destination}"
    if command:
        parts.append(f"rerun {command}")
    return "; ".join(parts)
```
