# P0.270 iOS Deploy Runbook Resource Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make iOS deploy runbook and final showcase operator actions
validation-aware for final resource handoff.

**Architecture:** Reuse the shared `add_final_resource_validation_command()`
helper after existing action collection. This keeps source reports responsible
for their own status while ensuring operator actions include a validation
command.

**Tech Stack:** Python, FastAPI backend report builders, pytest, ruff.
