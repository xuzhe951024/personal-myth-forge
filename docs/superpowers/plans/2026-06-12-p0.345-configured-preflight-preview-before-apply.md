# P0.345 Configured Preflight Preview Before Apply Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `final-configured-preflight` surface the read-only apply preview before direct final resource writer actions.

**Architecture:** Replace the configured-preflight provider-missing direct apply action with `make final-resource-apply-preview`, then add a post-dedupe filter that removes exact `make final-apply-resources` roots whenever the preview target is present.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, ruff.

- [x] **Step 4: Add helpers**

Add after `_dedupe()`:

```python
def _prefer_apply_preview_before_apply(actions: list[str]) -> list[str]:
    action_roots = {_operator_action_root(action) for action in actions}
    if FINAL_RESOURCE_APPLY_PREVIEW_ACTION not in action_roots:
        return actions
    return [
        action
        for action in actions
        if _operator_action_root(action) != FINAL_RESOURCE_APPLY_ACTION
    ]


def _operator_action_root(action: str) -> str:
    return action.split(" | ", 1)[0].strip()
```
