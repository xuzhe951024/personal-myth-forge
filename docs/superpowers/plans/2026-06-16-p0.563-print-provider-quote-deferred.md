# P0.563 Print Provider Quote Deferred Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the local print handoff become ready without configured Treatstock quote evidence while preserving the Treatstock provider path as the stricter configured mode.

**Architecture:** Keep the behavior inside `print_fulfillment_readiness.py`. The report will always include the configured Treatstock checks, but when `Settings.print_provider == "local"` those checks are emitted as `deferred` and ignored by blocker/action aggregation. `final_showcase_readiness.py` only needs copy and regression coverage because it already consumes the child print report status.

**Tech Stack:** Python 3.10, pytest, existing FastAPI backend readiness reports, Make targets.

---
