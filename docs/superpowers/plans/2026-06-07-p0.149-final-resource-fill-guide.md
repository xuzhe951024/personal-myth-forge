# P0.149 Final Resource Fill Guide Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an unambiguous final resource fill guide so the user can provide all API/key/iOS resources in one final handoff without confusing input source files and applied write destinations.

**Architecture:** Extend final resource requirement rows with explicit `input_source`, `write_destination`, `apply_command`, and `fill_action` fields. Add a read-only fill guide report builder, CLI, Make target, and final-local-report-refresh integration.

**Tech Stack:** Python backend report builders, pytest, existing Makefile, existing final resource preflight/requirements reports.
