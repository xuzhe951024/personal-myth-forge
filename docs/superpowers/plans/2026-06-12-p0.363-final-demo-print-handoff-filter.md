> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Keep the P0.362 selector, but split marker intent: live provider markers can match provider/live commands, while print handoff markers must indicate Treatstock or print quote handoff semantics. The selector remains local to `final_demo_launch.py`.

**Tech Stack:** Python 3.10, pytest, ruff, existing final demo launch report builder.
