# P0.385 iOS Rehearsal Next Action Command Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Keep `ios-device-launch-rehearsal` structured `next_action.command` values copy-pasteable while preserving detailed text in operator actions and blocker detail.

**Architecture:** `_next_action()` already chooses the first actionable top-level operator action. Add a small helper that strips detail suffixes only for the structured command field, leaving the operator action list unchanged.

**Tech Stack:** Python report builder, pytest.
