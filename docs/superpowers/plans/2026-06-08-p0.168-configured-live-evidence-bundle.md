# Configured Live Evidence Bundle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a read-only configured live evidence bundle report that packages final configured run commands, evidence files, current blocker, and safety policy into one acceptance handoff artifact.

**Architecture:** Add a new backend report builder that composes `final_configured_evidence_plan` and `live_provider_evidence`. Expose it through the existing CLI/Make pattern and keep all behavior read-only except writing the requested output JSON file.

**Tech Stack:** Python 3.10, FastAPI service package, pytest, ruff, Makefile.
