# P0.408 iOS Certificate Device Bundle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a read-only device action bundle to the iOS device launch certificate report.

**Architecture:** The existing certificate report already builds canonical `device_gates`. The implementation derives a top-level `device_action_bundle` from those gates, leaving status and blocker selection unchanged.

**Tech Stack:** Python 3.10, pytest, existing FastAPI backend report code, Make targets.
