# P0.47 Local Scene Asset Handoff Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make local demo-generated 3D scene assets available to iOS as backend-served, SceneKit-loadable HTTP URLs.

**Architecture:** Keep provider metadata generation separate from request-specific public URL rewriting. The local provider emits logical local asset references; FastAPI response handling rewrites those references using the incoming request base URL and exposes a deterministic DAE scene route. Final acceptance verifies the API-visible handoff, not just source strings.

**Tech Stack:** FastAPI, Pydantic models, pytest/TestClient, SwiftPM contract tests, SceneKit-compatible DAE metadata.
