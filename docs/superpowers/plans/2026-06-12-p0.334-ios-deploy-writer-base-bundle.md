# P0.334 iOS Deploy Writer Base Bundle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the iOS deploy config writer reuse the committed bundle id so physical-device handoff only needs machine-specific values.

**Architecture:** Update the existing shell writer in place. Add tests around default bundle id resolution while keeping env and flag overrides authoritative.

**Tech Stack:** POSIX shell, pytest, existing Make targets.
