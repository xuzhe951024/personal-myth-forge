# P0.392 Showcase Provider Child Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make final showcase 3D/NPC live provider rows inherit concrete child actions from provider handoff through live provider evidence.

**Architecture:** Keep the projection in `myth_forge_api.live_provider_evidence`, because that report is the source consumed by final showcase. Final showcase should continue to read live evidence `next_action` metadata.

**Tech Stack:** Python 3.10, pytest, existing backend report builders.
