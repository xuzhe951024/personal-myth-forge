- [ ] **Step 2: Add a helper**

Add this helper near `_prefer_complete_provider_handoff_chain()`:

```python
def _prefer_validated_backend_device_demo_action(actions: list[str]) -> list[str]:
    has_validated_backend_demo = any(
        "make backend-device-demo" in action
        and f"rerun {BACKEND_DEVICE_DEMO_VALIDATION}" in action
        for action in actions
    )
    if not has_validated_backend_demo:
        return actions
    return [
        action
        for action in actions
        if action != BACKEND_DEVICE_DEMO_BARE_ACTION
    ]
```

Update the dedupe pipeline:
