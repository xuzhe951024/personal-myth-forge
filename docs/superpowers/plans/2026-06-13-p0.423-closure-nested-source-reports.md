# P0.423 Closure Nested Source Reports Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Preserve nested source reports inside `final_launch_closure_packet.source_reports` for final iPhone evidence traceability.

**Architecture:** `final_launch_closure_packet` already summarizes child reports through `_source_summary()`. Extend that helper to carry an existing `source_reports` dictionary alongside the existing `device_action_bundle`, relying on the existing closure packet sanitizer for redaction.

**Tech Stack:** Python 3.10, pytest, FastAPI backend package, ignored `.local` JSON evidence reports.
