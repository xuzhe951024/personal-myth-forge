# P0.107 Evaluation Make Targets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add project-root Make targets for producing local 3D and NPC Agent evaluation readiness reports.

**Architecture:** Keep evaluation execution operator-side and explicit. `Makefile` owns the one-command entry points; readiness modules expose those Make commands in sanitized report actions; existing final launch, handoff, demo script, and iOS deploy runbook consume the readiness reports unchanged.

**Tech Stack:** GNU Make, FastAPI backend Python tests, SwiftPM contract/source checks, static HTML verification.

```python
import subprocess
from pathlib import Path


def test_evaluation_make_targets_dry_run_expected_local_commands() -> None:
    repo_root = Path(__file__).resolve().parents[3]

    result = subprocess.run(
        ["make", "-n", "backend-evaluate-3d", "backend-evaluate-npc", "backend-evaluate-local"],
        cwd=repo_root,
        check=False,
        capture_output=True,
        text=True,
    )

    assert result.returncode == 0
    output = result.stdout
    assert "backend-evaluate-3d" in output
    assert "evaluate-3d" in output
    assert "--provider local" in output
    assert "--suite default-v0" in output
    assert "--output .local/3d-evaluation-local.json" in output
    assert "backend-evaluate-npc" in output
    assert "evaluate-npc" in output
    assert "--tick-steps 2" in output
    assert "--output .local/npc-evaluation-local.json" in output
    assert output.index("evaluate-3d") < output.rindex("evaluate-npc")
```

### Task 2: GREEN Make Target Implementation

**Files:**
- Modify: `Makefile`

- [ ] **Step 1: Add minimal Make targets**

```make
.PHONY: backend-evaluate-3d backend-evaluate-npc backend-evaluate-local

backend-evaluate-3d:
	cd services/backend && uv run python -m myth_forge_api.cli evaluate-3d --provider local --suite default-v0 --output .local/3d-evaluation-local.json

backend-evaluate-npc:
	cd services/backend && uv run python -m myth_forge_api.cli evaluate-npc --provider local --suite default-v0 --tick-steps 2 --output .local/npc-evaluation-local.json

backend-evaluate-local: backend-evaluate-3d backend-evaluate-npc
```
