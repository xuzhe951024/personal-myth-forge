# P0.346 Demo Launch Preview Before Apply Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make configured final demo launch surface the read-only apply preview before direct final resource writer actions.

**Architecture:** Keep the final resource writer command in the commands list, but change the configured missing-resource launch phase command to `make final-resource-apply-preview`. First blocker, next action, and top-level operator actions already derive from that phase, so the handoff moves without adding a second report-level filter.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, ruff.
