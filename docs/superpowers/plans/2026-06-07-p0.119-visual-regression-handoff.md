# P0.119 Visual Regression Handoff Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make visual regression evidence a required, iPhone-visible part of the final local rehearsal and handoff flow.

**Architecture:** Add a read-only backend readiness wrapper for the saved visual regression report, wire that saved report into existing final handoff and iOS launch rehearsal source lists, and expose the readiness object through final demo launch. Add Swift models and summary rows so the iPhone final launch panel shows ready, missing, stale, and blocked visual evidence without exposing unsafe text.

**Tech Stack:** Python 3.10, FastAPI CLI modules, pytest, Makefile, Swift Package Manager, Swift Codable models.
