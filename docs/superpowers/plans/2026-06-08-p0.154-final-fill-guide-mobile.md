# P0.154 Final Resource Fill Guide Mobile Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Surface the sanitized final resource fill guide in the final demo launch payload and iOS Final Launch panel.

**Architecture:** Reuse the existing backend fill-guide report as the source of truth, attach it to `final_demo_launch`, decode it in Swift, and render compact mobile rows. The change is read-only and does not add provider calls or resource writes.

**Tech Stack:** Python FastAPI report builders, pytest, Swift Codable models, Swift summary builder, SwiftUI, SwiftPM contract/project/compile checks.

---
