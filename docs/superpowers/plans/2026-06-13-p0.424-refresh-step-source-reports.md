> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Preserve child `source_reports` in final local report refresh step rows.

**Architecture:** `final_local_report_refresh._run_step()` creates a compact step summary from each child report. Add `source_reports` copying beside the existing `device_action_bundle` copying, give the refresh-generated safe Xcode evidence snapshot its own action bundle, then rely on the existing final report sanitizer.

**Tech Stack:** Python 3.10, pytest, FastAPI backend package, ignored `.local` JSON reports.

        "label": str(first_blocker["label"]),
        "status": str(first_blocker["status"]),
        "classification": str(first_blocker["classification"]),
        "command": str(first_blocker["command"]),
        "detail": str(first_blocker["detail"]),
        "manual": True,
        "provider_calls": False,

    }
    return {
        "id": "mobile_xcode_build_evidence_actions",
        "label": "Mobile Xcode Build Evidence Actions",
        "source_report": "mobile_xcode_build_evidence",
        "status": "blocked",
        "actions": [action],
        "first_action": action,

            "manual": 1,

            "global_actions": 0,

            "commands_run": False,
            "provider_calls": False,

            "writes_backend_env": False,
