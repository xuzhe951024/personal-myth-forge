# P0.382 Final Showcase Capability Next Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add structured `next_action` objects to non-ready final showcase capability rows and display them in the iPhone Final Launch summary.

**Architecture:** Backend derives capability-level next actions in the common `_capability()` helper so all existing capability builders get the same contract. Swift extends `FinalShowcaseReadinessCapability` with an optional nested action and appends the action command and validation command to the capability summary row.

**Tech Stack:** Python report builder, pytest, Swift Codable models, SwiftPM contract tests.
