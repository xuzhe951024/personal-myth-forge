# P0.158 Local Showcase Smoke Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a no-key, HTTP-boundary local showcase smoke report that proves the iPhone-facing capture -> 3D generation -> NPC autonomy -> print quote loop can run locally.

**Architecture:** Create a backend `local_showcase_smoke_report` builder that uses FastAPI `TestClient` and temporary storage to run real REST calls without starting a server. Expose it through CLI/Make, embed it in `final_demo_launch`, decode and render compact rows on iOS, and register acceptance plus visual regression gates.

**Tech Stack:** FastAPI TestClient, Python report builders/tests, Make/CLI, Swift Codable/mobile summary tests, static 390x844 visual regression artifact.
