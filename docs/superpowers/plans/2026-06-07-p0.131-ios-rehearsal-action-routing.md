# P0.131 iOS Rehearsal Action Routing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the iOS device launch rehearsal report show concrete next actions after a successful safe refresh instead of telling the operator to rerun the same rehearsal command.

**Architecture:** Keep the safe shell wrapper and report source list unchanged. Enrich saved report summaries with bounded nested `operator_actions`, then make top-level action routing depend on whether reports are missing, blocked, partial, or ready.

**Tech Stack:** Python report builder, pytest, Makefile smoke verification, ignored `.local` JSON reports.

def _bounded_operator_actions(raw_actions: Any) -> list[str]:
    if not isinstance(raw_actions, list):
        return []
    actions: list[str] = []
    for action in raw_actions:
        if isinstance(action, str):
            actions.append(_clean_action(action))
        if len(actions) == 4:
            break
    return [action for action in actions if action]

- [x] **Step 4: Replace `_operator_actions` routing**

Replace the current `_operator_actions` function with:

```python
def _operator_actions(sequence: list[dict[str, Any]]) -> list[str]:
    if any(step["status"] == "missing" for step in sequence):
        actions = ["run make ios-device-launch-rehearsal"]
        for step in sequence:
            if step["status"] == "missing":
                actions.append(f"refresh {step['id']}: {step['command']}")
        return _dedupe(actions)

    actions: list[str] = []
    for step in sequence:
        if step["status"] != "blocked":
            continue
        nested_actions = step.get("operator_actions")
        if isinstance(nested_actions, list) and nested_actions:
            actions.extend(f"{step['id']}: {action}" for action in nested_actions)
        else:
            actions.append(f"review {step['id']}: {step['command']}")

    if actions:
        return _dedupe(actions)

    if all(step["status"] not in {"missing", "blocked"} for step in sequence):
        actions.append("continue with make backend-device-demo")
        actions.append("run make mobile-deploy-preflight after backend is running")
    return _dedupe(actions)
```
