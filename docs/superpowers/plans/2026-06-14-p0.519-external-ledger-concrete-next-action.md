- [ ] **Step 1: Add idempotent command helper**

Add this helper near `_next_action()` or the other operator-action helpers:

```python
def _command_with_validation(command: str, validation_command: str) -> str:
    if not command or not validation_command:
        return command
    validation_suffix = f"; rerun {validation_command}"
    if validation_suffix in command:
        return command
    return f"{command}{validation_suffix}"
```

- [ ] **Step 2: Use helper in promoted device blocker**

In `_promoted_device_blocker()`, compute `validation_command` before building `blocker`, then set:

```python
"command": _command_with_validation(command, validation_command),
```

Keep `validation_command` in the returned blocker when present.
