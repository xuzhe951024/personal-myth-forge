> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a read-only `ios-device-launch-certificate` report that tells an operator whether the checkout is ready to start the iPhone demo sequence.

**Architecture:** Create a small backend report builder that composes existing `final_handoff_index`, `ios_deploy_runbook`, and `final_demo_launch` reports, plus sanitized iOS deployment config facts. Expose it through the existing CLI and Makefile, then extend source gates, docs, and visual regression evidence.

**Tech Stack:** Python 3.10, FastAPI backend package, pytest, Makefile, Swift source checks.
