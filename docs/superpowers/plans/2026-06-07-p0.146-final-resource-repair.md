> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a safe repo-local repair command that clears stale final-resource placeholders without leaking or overwriting real user values.

**Architecture:** Implement a small Python report/apply module for final resource repair, expose it through the existing CLI and Makefile, then route existing-file init guidance to the new preview command. Keep all writes limited to the ignored final resources intake file and reuse existing report-output conventions.

**Tech Stack:** Python 3.10, pytest, FastAPI backend CLI conventions, POSIX shell init script, Makefile targets.
