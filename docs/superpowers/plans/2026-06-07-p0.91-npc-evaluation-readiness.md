DEFAULT_NPC_EVALUATION_PATH = Path("services/backend/.local/npc-evaluation-local.json")

Required API:

```python
@dataclass(frozen=True)
class NPCAgentEvaluationReadinessResult:
    exit_code: int
    report: dict[str, Any]


def build_npc_agent_evaluation_readiness_report(
    *,
    repo_root: Path | str | None = None,
    evaluation_file: Path | str | None = None,
) -> NPCAgentEvaluationReadinessResult:
    selected_repo_root = Path(repo_root) if repo_root is not None else _default_repo_root()
    selected_evaluation_file = (
        Path(evaluation_file)
        if evaluation_file is not None
        else selected_repo_root / DEFAULT_NPC_EVALUATION_PATH
    )
    # Read, validate, classify, sanitize, and return the report.
```

Report rules:

- Missing file returns `missing`, empty summary/coverage, no blockers, one operator action.
- Invalid JSON, non-object, wrong `kind`, `failed > 0`, `succeeded <= 0`, or missing coverage returns `blocked`.
- Ready report returns `ready`.
- Summary keys: `total_cases`, `succeeded`, `failed`, `tick_steps`.
- Coverage keys: `expected_npc_sets`, `trace_sets`, `proposed_action_plan_matches`, `tick_steps_completed`, `world_resolution_steps`.
- Blocker fields: `id`, `label`, `status`, `classification`, `command`, `detail`.
- Safety flags: `commands_run=False`, `provider_calls=False`, `global_mutation=False`, `provider_secrets_in_report=False`, `raw_private_context_in_report=False`, `raw_media_in_report=False`, `local_paths_in_report=False`, `payment_links_in_report=False`.
