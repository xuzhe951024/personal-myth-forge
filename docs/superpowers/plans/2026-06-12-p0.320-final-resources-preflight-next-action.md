# P0.320 Final Resources Preflight Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make final resources preflight reports expose a concrete first blocker and next action for API/key and iOS deploy resource handoff.

**Architecture:** Reuse existing parsed item and validation metadata. Add helper functions that choose the first blocked input, convert it into a blocker object, and mirror it into `next_action`.

**Tech Stack:** Python backend report builder, pytest, Makefile/CLI verification, SwiftPM compile checks.
