# P0.378 iOS Device Next Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show backend-provided child next actions for blocked iOS device actions in the iPhone Final Launch summary.

**Architecture:** Reuse the existing `FinalShowcaseReadinessNextAction` model as an optional nested field on `FinalShowcaseDeviceAction`. Render the nested command and detail through the existing `showcaseDeviceActionRow` builder.

**Tech Stack:** Swift Codable with `convertFromSnakeCase`, Swift Package Manager, existing mobile core contract test executable.
