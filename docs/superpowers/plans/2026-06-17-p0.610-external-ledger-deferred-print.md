# P0.610 External Ledger Deferred Print Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the final external action ledger point to the concrete print readiness blocker instead of hard-coding configured Treatstock quote when print fulfillment is blocked.

**Architecture:** Keep `print-fulfillment-readiness` as the source of truth. Build the ledger print action from its `first_blocker`/`next_action`, while preserving live/cost behavior for real print-provider quote calls.

**Tech Stack:** FastAPI backend report builder, pytest, project Make targets.
