# P0.544 Print Quote Auto Prerequisite Action Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make print fulfillment readiness point to the missing auto request prerequisite before recommending the configured print quote request writer.

**Architecture:** Add a small local handoff state helper inside `print_fulfillment_readiness.py` and mirror the iPhone-reachable URL predicate in `cli.py`. Existing checks continue to own report shape; only the configured quote request command/classification/detail changes when the request file is missing. Operator action ranking also treats these prerequisites as guarded print quote request handoffs so real quote calls do not outrank missing prerequisites.

**Tech Stack:** Python 3.10, FastAPI backend CLI helpers, pytest, existing Make safe wrappers.
