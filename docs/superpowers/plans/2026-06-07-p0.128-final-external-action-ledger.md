# P0.128 Final External Action Ledger Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a read-only final external action ledger that separates missing secrets, safe local writes, live provider cost gates, global Mac/Xcode actions, and device runtime actions.

**Architecture:** Create a focused backend report builder that composes existing final resource, apply preview, live provider, print fulfillment, iOS runbook, and showcase readiness reports. Wire it to CLI/Make and embed it into `final_demo_launch_report`, with source acceptance and docs covering the new surface.

**Tech Stack:** Python 3.10, pytest, existing FastAPI backend CLI/report builders, Makefile, existing source acceptance scanner.
