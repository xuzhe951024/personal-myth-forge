## Task 2: Swift Core GREEN

**Files:**
- Create: `apps/mobile/ios/Sources/PersonalMythForgeMobileCore/CaptureGenerationReceipt.swift`

- [ ] **Step 1: Add the receipt model**

Implement:

```swift
public enum CaptureGenerationReceiptStatus: String, Codable, Equatable, Sendable {
    case waiting
    case ready
    case needsAttention
}

public struct CaptureGenerationReceipt: Codable, Equatable, Sendable {
    public var status: CaptureGenerationReceiptStatus
    public var title: String
    public var detail: String
    public var rows: [String]
    public var privacyNotes: [String]
}
```

- [ ] **Step 2: Add the builder**

Implement:

```swift
public enum CaptureGenerationReceiptBuilder {
    public static func build(
        capture: ObjectCapture?,
        session: MythSession?,
        captureGenerationReadiness: CaptureGenerationReadiness
    ) -> CaptureGenerationReceipt
}
```

Rules:

- no capture: waiting with readiness title/detail
- capture without session: waiting with capture id, mode, media counts
- capture plus provenance: ready with provider, input mode, route, source counts
- capture plus no provenance: needs attention
- sanitize all row/detail text
