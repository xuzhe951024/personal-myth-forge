# P0.127 iOS Deploy Apply Preview Gate Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the iOS deploy runbook and device launch certificate route through `make final-resource-apply-preview` before `make final-apply-resources`.

**Architecture:** Reuse `build_final_resource_apply_preview_report` inside `ios_deploy_runbook.py`, add a required input slot and command sequence step, and rely on the existing device launch certificate summary to expose the new runbook step. The change is report-only and keeps all writer scripts/operator commands out of process.

**Tech Stack:** Python 3.10, pytest, existing FastAPI backend CLI/report builders, existing Swift model compatibility through unchanged runbook shape.
