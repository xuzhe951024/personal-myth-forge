> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an `auto` backend URL path for iPhone deploy config so the local device demo can write a LAN backend URL without manually typing the Mac IP.

**Architecture:** Extend the existing shell writer with a small auto-resolution helper that returns `http://<host>:<port>` before the current validation path. Add a Make alias for the common auto path and lock the behavior with deterministic tests that force the resolved host through environment variables.

**Tech Stack:** POSIX shell, Python stdlib for local IP discovery, Make, pytest.

- [x] **Step 1: Add auto URL resolution**

Add a `resolve_auto_backend_url` helper before validation. It should:

```sh
resolve_auto_backend_url() {
  port="${PMF_BACKEND_PORT:-8080}"
  if [ -n "${PMF_BACKEND_AUTO_HOST:-}" ]; then
    printf 'http://%s:%s' "$PMF_BACKEND_AUTO_HOST" "$port"
    return 0
  fi
  if ! command -v python3 >/dev/null 2>&1; then
    return 1
  fi
  PMF_BACKEND_PORT="$port" python3 - <<'PY'
import os
import socket
import sys

port = os.environ.get("PMF_BACKEND_PORT", "8080")
hosts = []
for target in [("8.8.8.8", 80), ("1.1.1.1", 80), ("224.0.0.1", 1)]:
    sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
    try:
        sock.connect(target)
        host = sock.getsockname()[0]
        if host and not host.startswith("127.") and host != "0.0.0.0":
            hosts.append(host)
    except OSError:
        pass
    finally:
        sock.close()
try:
    for host in socket.gethostbyname_ex(socket.gethostname())[2]:
        if host and not host.startswith("127.") and host != "0.0.0.0":
            hosts.append(host)
except OSError:
    pass
for host in hosts:
    print(f"http://{host}:{port}")
    sys.exit(0)
sys.exit(1)
PY
}
```

After parsing arguments and before missing-value checks, resolve when `backend_url` equals `auto`:

```sh
if [ "$backend_url" = "auto" ]; then
  resolved_backend_url=$(resolve_auto_backend_url || true)
  if [ -z "$resolved_backend_url" ]; then
    printf '%s\n' "Could not auto-detect a non-loopback backend host." >&2
    missing=1
  else
    backend_url="$resolved_backend_url"
    printf '%s\n' "Auto backend URL: $backend_url"
  fi
fi
```
