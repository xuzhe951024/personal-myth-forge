Add helpers:

```python
def _dedupe_operator_actions(actions: list[str]) -> list[str]:
    normalized = _dedupe(actions)
    validation_aware = {
        action.split("; rerun ", 1)[0].strip()
        for action in normalized
        if "; rerun " in action
    }
    return [
        action
        for action in normalized
        if action not in validation_aware
    ]
```

Then use `_dedupe_operator_actions()` in `_operator_checklist()` and `_operator_actions()`.
