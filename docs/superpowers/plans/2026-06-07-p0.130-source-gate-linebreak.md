# P0.130 Source Gate Linebreak Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the iOS showcase source acceptance gate robust to harmless Swift line breaks so final local rehearsal no longer fails hard after P0.129.

**Architecture:** Extend the existing lightweight source scanner with multi-token requirements that check several strings in the same file. Use that for the mobile external action ledger handoff instead of forcing a single adjacent `externalActionLedgerRows(from:` source string.

**Tech Stack:** Python dataclasses, pytest, Swift source acceptance fixtures, Makefile rehearsal smoke command.

### Task 2: Implement Multi-Token Source Requirements

**Files:**

- [x] **Step 1: Extend `SourceRequirement`**

Change the dataclass to:

```python
@dataclass(frozen=True)
class SourceRequirement:
    file: str
    contains: str
    all_contains: tuple[str, ...] = ()
```

- [x] **Step 2: Add matching helpers**

Add helpers near `run_ios_showcase_acceptance`:

```python
def _requirement_matches(text: str, requirement: SourceRequirement) -> bool:
    if requirement.all_contains:
        return all(token in text for token in requirement.all_contains)
    return requirement.contains in text


def _requirement_evidence(requirement: SourceRequirement) -> str:
    if requirement.all_contains:
        return " && ".join(requirement.all_contains)
    return requirement.contains
```

- [x] **Step 3: Use helpers in the scanner**

Where source requirements are evaluated, replace direct
`requirement.contains in text` checks with `_requirement_matches(text, requirement)`.
Use `_requirement_evidence(requirement)` when writing evidence and missing
entries.

- [x] **Step 4: Update the mobile external action ledger requirement**

Replace the single brittle requirement:

```python
SourceRequirement(
    "apps/mobile/ios/Sources/PersonalMythForgeMobileCore/FinalLaunchMobileSummary.swift",
    "externalActionLedgerRows(from:",
),
```

with:

```python
SourceRequirement(
    "apps/mobile/ios/Sources/PersonalMythForgeMobileCore/FinalLaunchMobileSummary.swift",
    "externalActionLedgerRows && from: report.finalExternalActionLedger",
    (
        "externalActionLedgerRows",
        "from: report.finalExternalActionLedger",
    ),
),
```
