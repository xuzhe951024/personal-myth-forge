# P0.19 Guided Scan Entry Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a native iOS guided scan entry that feeds Object Capture photos into the existing capture-to-myth upload path.

**Architecture:** Introduce `guided_scan` as a capture-mode contract across backend and mobile core, add a pure Swift importer for captured scan photos, then wire a SwiftUI `ObjectCaptureSession` sheet behind iOS-only compile guards. The current SwiftPM gates keep compiling on macOS; the P0.18 Xcode gate will cover the iOS-only path after the Apple SDK license is accepted.

**Tech Stack:** FastAPI, Pydantic, Swift 6, SwiftPM, SwiftUI, RealityKit Object Capture, Xcode project files.

---

Do not run `xcode-select`, simulator launch/install commands, device install
commands, archive commands, signing setup, or `sudo xcodebuild -license`.
