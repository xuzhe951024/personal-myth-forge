# P0.102 3D Generation Input Review Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an iPhone pre-forge receipt that explains how the current capture input will become a 3D provider request.

**Architecture:** Add a pure Swift mobile-core builder for `ThreeDGenerationInputReview`, then render it in `CaptureFormView` with state owned by `ForgeRootView`. Extend existing source gates and visual regression evidence so the receipt becomes part of the required iOS showcase.

**Tech Stack:** Swift 6, SwiftUI, SwiftPM executable contract tests, FastAPI source-gate tests, static HTML visual regression.
