- [ ] **Step 1: Add wrapper scripts**

`write_final_acceptance_local.sh`:

```sh
#!/usr/bin/env sh
set -eu

mkdir -p services/backend/.local
set +e
(
  cd services/backend
  uv run python -m myth_forge_api.cli final-acceptance \
    --profile quick \
    --provider-mode local \
    --repo-root ../.. \
    --output .local/final-acceptance-local.json
)
status=$?
set -e

if [ "$status" -eq 0 ] || [ "$status" -eq 2 ]; then
  printf '%s\n' "accepted final acceptance exit code $status"
  printf '%s\n' "wrote services/backend/.local/final-acceptance-local.json"
  exit 0
fi

printf '%s\n' "final acceptance failed before writing a usable report: exit $status" >&2
exit "$status"
```

`write_ios_deploy_runbook_local.sh` follows the same pattern for:

```sh
uv run python -m myth_forge_api.cli ios-deploy-runbook \
  --mode local \
  --repo-root ../.. \
  --output .local/ios-deploy-runbook-local.json
```

- [ ] **Step 2: Add Make targets**
