# P0.73 Mobile Context Capsule Review Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Require an explicit, privacy-preserving context capsule approval before the iPhone app can forge a myth session.

**Architecture:** Add a Swift core review builder for manually entered context summaries. Wire a SwiftUI review panel and approval toggle into the capture form, with root-owned approval state reset on edits and a guard in `forgeMyth`.

**Tech Stack:** SwiftPM iOS core, SwiftUI app source, FastAPI source acceptance, static HTML visual evidence.
