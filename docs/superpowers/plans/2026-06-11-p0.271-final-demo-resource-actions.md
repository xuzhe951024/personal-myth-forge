# P0.271 Final Demo Resource Actions Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make final demo launch operator actions use `make final-resources-preflight` for every selected `final-resources.env` handoff action.

**Architecture:** Canonicalize final resource validation in the shared `operator_actions` helper, then apply that helper to final demo launch top-level operator actions before dedupe/truncation. This keeps final demo launch status and first-blocker behavior unchanged while making resource handoff actions self-validating.

**Tech Stack:** Python, FastAPI backend report builders, pytest, ruff.
