# P0.78 Mobile Final Launch Mode Regression

Date: 2026-06-06

Scope:

- Added `PMF_FINAL_LAUNCH_MODE` to final resource preflight, final resource apply, iOS deployment templates, and project-local deploy config writer.
- Added mobile `FinalLaunchMode` model, `PMFFinalLaunchMode` plist handoff, and a `Local` / `Configured` segmented control in `ForgeRootView`.
- Switched the final launch report load path from hard-coded local mode to `finalLaunchMode.rawValue`.
- Updated mobile source gates so the iPhone can inspect configured readiness without running commands or live providers.

Safety notes:

- Unsupported mode values are blocked by preflight/apply and default to local in iOS config reading.
- Switching modes on iPhone only reloads `GET /v1/final-demo-launch?mode=<mode>`.
- No provider secrets, raw media, local paths, payment links, Xcode commands, signing commands, or global mutations are introduced by this slice.
