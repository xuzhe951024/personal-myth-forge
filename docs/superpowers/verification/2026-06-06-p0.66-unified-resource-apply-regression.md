# P0.66 Unified Final Resource Apply Regression

Date: 2026-06-06

## Scope

P0.66 adds a single final resource apply path for configured demo handoff:

- tracked template: `services/backend/final-resources.env.example`
- ignored filled copy: `services/backend/.local/final-resources.env`
- apply script: `services/backend/scripts/apply_final_resources.sh`
- Make target: `make final-apply-resources`

The apply script validates required Meshy/OpenAI/iOS values, supports optional
Treatstock print quote handoff, rejects unsupported or unsafe inputs before
writing, and delegates to the existing backend/iOS config writers.

## Safety

The apply path writes only ignored project-local files:

- `services/backend/.env`
- `apps/mobile/ios/Config/Deployment.local.xcconfig`

It does not call providers, start servers, run Xcode, sign apps, touch keychain,
accept Apple licenses, write global config, or print secret values.
