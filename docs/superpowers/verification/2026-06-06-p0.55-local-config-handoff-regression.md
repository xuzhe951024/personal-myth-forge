# P0.55 Local Config Handoff Regression

Date: 2026-06-06

## Scope

P0.55 adds a project-local writer for the ignored iOS device deployment config:

```bash
DEVELOPMENT_TEAM=ABCDE12345 \
PRODUCT_BUNDLE_IDENTIFIER=com.example.personalmythforge \
PMF_BACKEND_BASE_URL=http://192.168.1.10:8080 \
make mobile-write-deploy-config
```

The writer creates `apps/mobile/ios/Config/Deployment.local.xcconfig` and then
hands off to the existing validation command:

```bash
make mobile-deploy-preflight
```

## Safety Boundary

- The writer only writes the ignored local xcconfig.
- It refuses tracked `Deployment.local.xcconfig`.
- It rejects loopback backend URLs for physical iPhone demos.
- It does not call Xcode signing, keychain, Apple developer portal, Meshy,
  OpenAI, or print provider services.
