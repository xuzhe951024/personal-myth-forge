# P0.75 Mobile NPC Agent Mode Regression

## Scope

P0.75 adds a mobile `NPC Agent Mode` panel above the existing NPC autonomy
controls. It tells the iPhone operator whether NPC behavior is using local demo
logic, OpenAI-backed AI Agent logic, or is blocked by backend setup.

## Behavior Covered

- Swift core builds a deterministic `NPCAgentModeSummary`.
- No session maps to a waiting state.
- `local_agent_runtime` maps to local demo mode.
- `openai_structured_runtime` and OpenAI provider readiness map to AI-ready
  mode.
- Missing `OPENAI_API_KEY` maps to setup-needed mode.
- Latest NPC tick runtime and traces override older session runtime details.
- Display copy redacts provider secrets, authorization headers, checkout fields,
  local capture refs, and full local paths.
- `ForgeRootView` renders `NPCAgentModeView` before the NPC tick controls.
- iOS showcase acceptance requires the new builder, SwiftUI view, root wiring,
  and contract test evidence.
