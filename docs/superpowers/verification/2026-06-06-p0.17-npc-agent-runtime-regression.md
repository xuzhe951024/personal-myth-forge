# P0.17 NPC Agent Runtime Regression

Date: 2026-06-06

## Scope

P0.17 verifies that myth sessions can include an NPC agent runtime trace with
one belief/intention/action proposal per NPC, while preserving the existing
`npc_reactions` and `world_resolution` contracts.

No simulator, physical iPhone, Xcode signing, ARKit runtime, Unity movement
execution, voice NPC runtime, real OpenAI key run, or long-term NPC memory was
executed.

## Acceptance Notes

- Backend `MythSession` includes `npc_agent_runtime` and `npc_agent_traces`.
- Local NPC output now forms trace rows before deriving visible reactions.
- OpenAI structured output can carry trace rows and falls back to synthesized
  traces from valid reactions when traces are missing.
- Swift `MythSession` decodes older responses without trace fields as empty
  defaults.
- `NPCReactionsView` renders compact trace rows when traces are present.
