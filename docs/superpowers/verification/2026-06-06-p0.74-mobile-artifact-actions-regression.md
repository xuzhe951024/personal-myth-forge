# P0.74 Mobile Artifact Actions Regression

## Scope

P0.74 adds a mobile `Artifact Handoff` action panel below the 3D artifact
preview. It turns prepared generated-asset state into SceneKit, share,
conversion, retry, and waiting actions.

## Behavior Covered

- Swift core builds deterministic action summaries from `PreparedArtifactAsset`.
- No session maps to a waiting state.
- SceneKit-ready USDZ/USD/DAE/SCN assets expose open and share actions.
- Cached GLB/GLTF exposes share plus a conversion-required warning.
- Download failure exposes a retry action.
- Action copy redacts provider secrets, checkout fields, and full local paths.
- The iOS app renders `ArtifactHandoffActionsView` from `Artifact3DPreviewView`.
- iOS showcase acceptance requires the new core builder, SwiftUI view, and
  contract test evidence.
