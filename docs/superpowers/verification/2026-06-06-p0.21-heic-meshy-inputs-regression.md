# P0.21 HEIC Meshy Inputs Regression

Date: 2026-06-06

## Scope

P0.21 adds backend-side HEIC/HEIF-to-JPEG provider input preparation for iOS
capture flows. Upload manifests continue to record original HEIC/HEIF media;
`POST /v1/myth-sessions/from-capture` converts decodable HEIC/HEIF reference
images into backend-only JPEG data URIs before 3D provider dispatch.

## Security And Privacy Checks

- Provider keys remain backend-only.
- HEIC/HEIF source bytes and JPEG data URIs are not returned in API responses.
- Capture manifests keep original content type and `local-capture://` URI.
- Invalid HEIC/HEIF decode errors return sanitized 422 responses.
- Local filesystem paths are redacted from provider-facing error responses.
