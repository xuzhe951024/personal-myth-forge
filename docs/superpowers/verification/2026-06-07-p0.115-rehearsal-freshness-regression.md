# P0.115 Rehearsal Freshness Regression

P0.115 adds a git-HEAD freshness guard to
`ios_device_launch_rehearsal_readiness`.

Behavior covered:

- Backend field: `ios_device_launch_rehearsal_readiness.freshness`
- Stale blocker: `ios_device_launch_rehearsal_freshness`
- Operator action: `rerun make ios-device-launch-rehearsal`
- Mobile row: `Freshness: stale_report; rerun iOS device launch rehearsal.`
