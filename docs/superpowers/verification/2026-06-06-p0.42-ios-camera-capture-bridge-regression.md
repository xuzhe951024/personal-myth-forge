# P0.42 iOS Camera Capture Bridge Regression

## Scope

P0.42 adds a direct iOS `Take Photo` bridge for single-photo object capture.
The app converts captured camera output into JPEG bytes and feeds it through the
same `CaptureMediaSelection(mode: .singlePhoto)` contract used by the existing
PhotosPicker path.

## Remaining Blockers

P0.42 does not claim real iPhone camera validation. Final device acceptance still
requires:

- local `Deployment.local.xcconfig` with Team ID, bundle id, and LAN backend URL
- Apple SDK license accepted outside Codex
- a signed iPhone build
- real camera permission and `UIImagePickerController` runtime validation
- live Meshy/OpenAI provider-key runs when keys are provided
