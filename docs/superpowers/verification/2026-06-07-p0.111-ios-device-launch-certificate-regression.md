# P0.111 iOS Device Launch Certificate Regression

P0.111 adds a read-only Mac-side certificate for starting the final physical
iPhone demo sequence.

## Scope

- `myth_forge_api.ios_device_launch_certificate`
- `myth_forge_api.cli ios-device-launch-certificate`
- `make ios-device-launch-certificate`
- `services/backend/.local/ios-device-launch-certificate.json`

configured handoff inputs are fully ready. The command still writes a sanitized
blocked certificate to `services/backend/.local/ios-device-launch-certificate.json`.

## Safety

The certificate is read-only. It does not call providers, start servers, run
deploy preflight, run Xcode, touch signing/keychain state, accept Apple licenses,
or write backend/iOS config files.
