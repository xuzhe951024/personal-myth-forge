Date: 2026-06-06

## Scope

P0.32 adds one final acceptance CLI report:

```bash
cd services/backend
uv run python -m myth_forge_api.cli final-acceptance \
  --profile quick \
  --provider-mode local \
  --output /tmp/personal-myth-forge-final-acceptance.json
```

The command gathers:

- provider handoff readiness
- demo acceptance
- optional full backend/Swift regression checks
- iOS deploy preflight
- Xcode build gate

Known local provider, iOS deploy, and Xcode environment gates are classified as
`blocked` with CLI exit `2`, not as product failures.

## Safety Review

- The final acceptance command does not mutate `.env`, process-global
  environment, Xcode selection, signing state, or Apple license state.
- Provider keys remain backend-only.
- Local quick mode makes no live Meshy/OpenAI calls.
- Configured strict mode exits before provider calls until core providers are
  real-provider-ready.
- Reports are sanitized for bearer tokens, API key values, data URIs, `raw=`
  payload markers, file URLs, and local filesystem paths.
- iOS signing/device gates remain explicit local operator actions.
