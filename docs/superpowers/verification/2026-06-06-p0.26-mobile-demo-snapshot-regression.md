## Safety Checks

- Snapshot model stores only `MythSession`, `NPCAgentTick`, schema version, and
  timestamp.
- Contract tests assert snapshot JSON does not contain sample raw media bytes,
  `Authorization=Bearer`, `api_key=`, or `file:///`.
- User-facing snapshot status view does not display local paths.
- Provider keys remain backend-only.
- This is not server-side NPC memory or a background autonomous loop.
