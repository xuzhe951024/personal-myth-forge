# P0.69 Final Launch Unified Apply Regression

## Scope

P0.69 updates the final demo launch report so resource application has one
primary operator path:

```bash
make final-apply-resources
```

The lower-level writer targets remain available for debugging, but
`final-demo-launch` local/configured command lists no longer advertise them as
the main final handoff.

Final launch report checks:

- phase id `apply_final_resources` exists
- phase command is `make final-apply-resources`
- old launch phase ids `write_backend_env` and `write_ios_deploy_config` are absent
- local mode marks the phase partial when iOS config is ready but final provider resources are not
- local/configured command lists do not include `backend-write-provider-env`
- local/configured command lists do not include `mobile-write-deploy-config`

## Safety

This iteration does not call Meshy, OpenAI, Treatstock, Sculpteo, Apple, or
Xcode services. It does not write secrets or global machine state. The launch
report remains sanitized for provider keys and local paths.
