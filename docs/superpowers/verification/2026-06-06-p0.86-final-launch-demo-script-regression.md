# P0.86 Final Launch Demo Script Regression

Date: 2026-06-06
Branch: `codex/p0.86-final-launch-demo-script`

## Scope

P0.86 makes the iPhone `Demo Script` final-launch aware. The local capture,
forge, 3D scene, NPC autonomy, print quote, and provider resource loop can still
complete, but the script now appends a `Final Launch` row when
`FinalLaunchMobileSummary` is available. The autopilot stops at `Check Launch`
when final launch readiness is blocked or waiting.

No backend endpoint, provider call, deploy command, signing config, or global
system state changed.
