# P0.80 Final Launch Mode Policy Summary Regression

## Scope

P0.80 makes the iPhone Final Launch Status panel explicitly show the currently
loaded launch report mode and live-call policy:

- `Mode: Local` or `Mode: Configured`
- `Live calls by default: no`
- `Consent flag: --allow-live-provider-calls`

The app remains read-only. It still does not execute final launch commands,
read provider keys, mutate resource files, or call live providers.
