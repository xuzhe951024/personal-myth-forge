# P0.70 Final Resources Preflight Regression

## Scope

P0.70 adds a read-only preflight for:

```text
services/backend/.local/final-resources.env
```

The preflight validates the unified resource file before
`make final-apply-resources` writes ignored backend and iOS config files.

## Behavior Covered

- Missing final resources file returns status `missing` and exit `2`.
- Unknown keys return status `blocked` without leaking values.
- Loopback `PMF_BACKEND_BASE_URL` returns status `blocked`.
- `PRINT_PROVIDER=treatstock` requires `TREATSTOCK_API_KEY`.
- Valid local-print resources return status `ready` and exit `0`.
- `final-demo-launch` includes `final_resources_preflight`.
- `apply_final_resources` phase reflects missing, blocked, or ready preflight.
- Source template acceptance requires the CLI command, Make target, and launch integration.

## Safety

The preflight does not write `services/backend/.env`, does not write
`Deployment.local.xcconfig`, does not call providers, does not run Xcode, and
does not mutate global machine state. Reports include key names and configured
status only; provider key values and local absolute paths stay out of JSON,
logs, docs, and screenshots.
