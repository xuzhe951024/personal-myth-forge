# P0.49 Mobile Demo Script Regression

## Scope

P0.49 adds an iOS `Demo Script` panel that derives the next visible demo action
from existing app state. The script covers:

1. Capture Object
2. Forge Myth
3. Load 3D Scene
4. Run NPC Autonomy
5. Request Print Quote
6. Check Resources

The feature is mobile-only source/app work. It does not call provider APIs,
create keys, change backend schemas, accept the Apple SDK license, or mutate
global Xcode configuration.

## Safety

The Swift builder and source acceptance protect the same privacy boundary:

- no raw capture media in script state
- no local file paths in script state
- no provider keys or bearer tokens in script state
- no checkout URLs in script state
- no live provider, device, simulator, Xcode, or Apple service calls in source
  acceptance
