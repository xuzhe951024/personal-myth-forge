# P0.156 iOS Device Evidence Bundle Visual Regression

P0.156 adds a default visual regression artifact for the iPhone final launch
status card after `ios_device_evidence_bundle` is decoded and rendered on
mobile.

Artifacts:

- HTML: `docs/superpowers/verification/p0.156-ios-device-evidence-bundle.html`
- PNG: `docs/superpowers/verification/assets/p0.156-ios-device-evidence-bundle-390x844.png`
- Viewport: `390x844`

The visual proof shows:

- `Final Launch Status`
- `Device Evidence`
- `iOS device evidence blocked`
- `mobile_deploy_preflight`
- `make mobile-deploy-preflight`
- `Safety: commands_run=false xcode=false global=false`

Safety:

- No provider calls were made.
- No deploy preflight, backend server, Xcode, signing, keychain, or global
  machine commands were executed.
- No backend `.env`, iOS deploy config, provider secrets, raw media, local
  filesystem paths, private LAN URLs, or commerce links are present in the
  visual fixture.
