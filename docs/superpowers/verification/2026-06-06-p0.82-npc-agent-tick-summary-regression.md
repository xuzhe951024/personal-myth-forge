# P0.82 NPC Agent Tick Summary Regression

## Scope

P0.82 adds a mobile-facing NPC Agent tick summary. The summary explains whether NPC Agents are waiting, running, showing initial session traces, or showing the latest resolved autonomous tick.

## Contract Coverage

- `NPCAgentTickSummaryBuilder` builds waiting, running, ready, and needs-attention states.
- The summary includes runtime label, tick label, decision label, saved tick count, trace count, world-resolution counts, and privacy notes.
- Unsafe text is redacted before summary values are encoded or rendered.
- `ForgeRootView` passes the summary into `NPCTickView`.
- `NPCTickView` renders the summary above existing tick action controls and trace detail cards.
- `mobile_npc_agent_tick_summary` is now part of backend iOS showcase acceptance.
