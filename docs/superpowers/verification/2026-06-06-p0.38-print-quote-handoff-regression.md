# P0.38 Print Quote Handoff Regression

Date: 2026-06-06

## Scope

P0.38 adds the backend print quote handoff contract, a deterministic local draft
quote provider path, print provider readiness config slots, and a
`print_quote_acceptance` check in final acceptance.

This iteration does not implement live Treatstock/Sculpteo quote calls, payment
checkout, print order submission, or mobile print review UI.
