# P0.47 Local Scene Asset Handoff Regression

## Scope

P0.47 makes deterministic local 3D output downloadable by the iOS preview path.
Local demo sessions now expose a backend-served `ios_scene_asset` DAE URL under:

```http
GET /v1/generated-assets/{session_id}/game.glb
GET /v1/generated-assets/{session_id}/scene.dae
```

The routes return deterministic local GLB metadata and Collada geometry. They do
not include raw capture media, provider keys, local filesystem paths, or personal
source text.

## Safety

The new local asset handoff acceptance confirms:

- Local scene asset URL is HTTP and backend-served.
- Primary local game asset URL is also downloadable.
- Scene asset content contains `COLLADA` and the session id.
- Report contains no `local://` URI, `data:image`, provider bearer token, `/tmp`,
  or `/Users/` path.
- No live provider calls.
- No global mutation.
