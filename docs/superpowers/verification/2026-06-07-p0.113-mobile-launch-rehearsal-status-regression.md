# P0.113 Mobile Launch Rehearsal Status Regression

P0.113 adds the saved iOS device launch rehearsal result to the existing final
launch readiness surface:

- Backend field: `ios_device_launch_rehearsal_readiness`
- Swift model: `IOSDeviceLaunchRehearsalReadinessReport`
- Mobile summary rows: `launchRehearsalRows`
- UI section: `Launch Rehearsal`

Verification commands:

```bash
cd services/backend
uv run pytest tests/test_final_demo_launch.py tests/test_mobile_final_launch_readiness_acceptance.py tests/test_ios_showcase_acceptance.py -q
cd ../..
swift run --package-path apps/mobile/ios PersonalMythForgeMobileCoreContractTests
swift run --package-path apps/mobile/ios PersonalMythForgeMobileProjectChecks
```
