# P0.19 Guided Scan Entry Regression

## Scope

P0.19 adds `guided_scan` to the backend/mobile capture contract and wires a
native iOS guided scan entry into the SwiftUI app shell. The guided scan path
imports Object Capture photos into the existing capture-to-myth upload flow; it
does not reconstruct a local mesh or install the app on a simulator/device.
