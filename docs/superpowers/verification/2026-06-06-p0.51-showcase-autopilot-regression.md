# P0.51 Showcase Autopilot Regression

## Scope

P0.51 adds a one-step, operator-controlled autopilot to the mobile `Demo Script`
panel:

1. `ShowcaseAutopilotPlanner` chooses the next safe action from current typed
   app state.
2. `DemoScriptView` renders a compact button and planner detail.
3. `ForgeRootView` dispatches the button to exactly one existing action:
   `forgeMyth`, `runAutonomy`, `advanceNPCTick`, or `requestPrintQuote`.
4. The planner disables execution while forge, autonomy, NPC advance, or quote
   loading is already in progress.
5. Unsafe detail text is redacted before display or JSON encoding.

The feature does not add a backend endpoint, live provider call, app-launch
automation, retry loop, Xcode signing change, Apple SDK license mutation, or
global environment change.

## Safety

- The planner stores no raw capture media.
- Unsafe local paths, provider keys, capture URLs, bearer text, and checkout text
  are redacted by the Swift builder.
- Source acceptance does not run live provider calls, device deployment,
  simulator boot, Xcode license acceptance, or global developer directory
  changes.
