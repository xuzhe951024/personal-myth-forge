# P0.34 Generation Provenance Contract Regression

Date: 2026-06-06

## Scope

P0.34 adds optional safe generation provenance to backend `GeneratedAsset`
responses, provider outputs, demo acceptance reports, and iOS decode/display.

The contract proves whether generation used text, one source image, or a
multi-image guided scan without returning raw capture media, provider data URIs,
local paths, or provider secrets.
