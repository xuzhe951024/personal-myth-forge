# P0.50 NPC Ritual Scene Regression

## Scope

P0.50 adds a native iOS showcase layer for spatial NPC action around the
generated artifact:

1. `NPCRitualSceneBuilder` derives three ritual actors from a `MythSession` and
   optional latest `NPCAgentTick`.
2. Latest tick data wins for runtime, world changes, stance, and action.
3. Sparse latest ticks are backfilled from the session so the preview keeps
   three visible NPCs.
4. `Artifact3DPreviewView` overlays SceneKit marker and ring nodes on either a
   downloaded scene asset or the fallback artifact proxy.
5. `ForgeRootView` passes `latestNPCTick` through `ArtifactSummaryView` into the
   3D preview.

The feature does not add Unity, voice NPCs, provider API calls, signing changes,
or Apple SDK/license mutations.

## Safety

- No raw capture media is encoded in ritual scene state.
- Unsafe local paths, provider keys, capture URLs, and checkout text are redacted
  by the Swift builder.
- Source acceptance does not run live provider calls, device deployment,
  simulator boot, Xcode license acceptance, or global developer directory
  changes.
