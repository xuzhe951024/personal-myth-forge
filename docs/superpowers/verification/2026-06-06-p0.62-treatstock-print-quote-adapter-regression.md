# P0.62 Treatstock Print Quote Adapter Regression

Date: 2026-06-06

## Scope

P0.62 adds the first live third-party print quote adapter path through
Treatstock and serves a local downloadable 3MF print candidate endpoint for demo
sessions.

This iteration does not place Treatstock orders, collect shipping/contact data,
collect payment data, upload private local files, implement Sculpteo, or make
live Treatstock calls in automated tests.

## Source Reference

The implementation follows the official Treatstock API documentation:

- API token/private-key requirement.
- `POST /api/v2/printable-packs` with `files-urls[]` and location data.
- `GET /api/v2/printable-pack-costs/` for material/cost results.
- `not_calculated_yet` retry behavior.
- `place-order/create` is separate order creation and is intentionally out of
  scope for P0.62.

Source: https://www.treatstock.com/help/article/83-api-documentation

## Implemented Checks

- `TreatstockPrintProvider` validates API key, supported STL/PLY/3MF formats,
  and HTTP(S) candidate URLs.
- The adapter POSTs URL uploads to Treatstock printable packs, optionally PUTs
  quantity updates, retries bounded `not_calculated_yet` cost responses, and
  selects the lowest returned price.
- `PRINT_PROVIDER=treatstock` is wired through settings, provider factory, and
  provider readiness.
- `/v1/print-quotes` sanitizes Treatstock provider failures before returning
  HTTP 502 details.
- `/v1/print-candidates/{session_id}/print.3mf` serves a local placeholder 3MF
  ZIP payload for demo print candidate handoff.
