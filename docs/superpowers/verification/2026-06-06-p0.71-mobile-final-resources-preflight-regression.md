# P0.71 Mobile Final Resources Preflight Regression

## Scope

P0.71 surfaces backend `final_resources_preflight` status in the iPhone Device
Preflight summary.

## Behavior Covered

- Swift decodes `FinalResourcesPreflightReport` from `FinalDemoLaunchReport`.
- Device Preflight includes item id `final_resources`.
- Missing final resources preflight maps to waiting.
- Ready final resources preflight maps to ready.
- Blocked final resources preflight maps to blocked.
- Unsafe detail strings are redacted before display.
- Backend source acceptance requires the model, item, and contract tests.

## Safety

The iPhone app still does not read provider keys, `.env`, or
`final-resources.env`. It consumes only the sanitized backend final launch report
and applies existing mobile redaction before rendering details.
