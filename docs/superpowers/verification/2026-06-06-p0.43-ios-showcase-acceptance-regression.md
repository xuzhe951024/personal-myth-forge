# P0.43 iOS Showcase Acceptance Regression

## Scope

P0.43 adds `ios_showcase_acceptance`, a source-only in-process final acceptance
check. It verifies that the checked-in iOS source still contains the complete
minimal showcase surface without running Xcode, SwiftPM, simulators, devices,
provider APIs, or network calls.

## Remaining Blockers

The final two blockers are unchanged:

- `mobile_deploy_preflight`: requires ignored local Team ID, bundle id, and
  iPhone-reachable backend URL
- `mobile_xcode_build`: requires Apple SDK license/build environment readiness

P0.43 does not claim signed iPhone deployment, live camera runtime validation,
or live Meshy/OpenAI provider runs.
