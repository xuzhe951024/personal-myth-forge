# P0.67 ARKit Scan Generation Acceptance Regression

Date: 2026-06-06

## Scope

P0.67 adds a final acceptance gate named
`arkit_scan_generation_acceptance`. It proves the ARKit/Object Capture path that
the iOS demo depends on:

- synthetic `arkit_scan` capture package stored through `LocalCaptureStore`
- one GLB scan asset plus two reference images
- backend source building through `build_capture_generation_sources(...)`
- normal myth-session pipeline through `create_demo_myth_session(...)`
- recording local 3D provider request with source images and source asset
- sanitized report with counts, content types, provenance, and no raw media
