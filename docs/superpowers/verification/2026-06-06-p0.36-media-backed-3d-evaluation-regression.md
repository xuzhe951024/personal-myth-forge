# P0.36 Media-Backed 3D Evaluation Regression

Date: 2026-06-06
Branch: `codex/p0.36-media-backed-3d-evaluation`

## Scope

P0.36 adds `evaluate-3d --suite guided-scan-smoke-v0`, a local no-key smoke
suite that sends synthetic PNG source images through the 3D provider contract.
The report records source image counts, source roles, and generation
provenance without writing raw media data URIs, local file paths, authorization
headers, or provider keys.

## Guided Suite Smoke

```bash
cd services/backend
uv run python -m myth_forge_api.cli evaluate-3d \
  --provider local \
  --suite guided-scan-smoke-v0 \
  --output /tmp/personal-myth-forge-p0.36-guided-scan-eval.json

python3 - <<'PY'
import json
from pathlib import Path
report = json.loads(Path('/tmp/personal-myth-forge-p0.36-guided-scan-eval.json').read_text())
assert report['suite'] == 'guided-scan-smoke-v0'
assert report['total_cases'] == 3
assert report['coverage']['input_modes']['single_image'] == 1
assert report['coverage']['input_modes']['multi_image'] == 2
assert 'data:image' not in json.dumps(report)

PY
```
