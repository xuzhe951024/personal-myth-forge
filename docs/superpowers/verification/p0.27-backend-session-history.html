Backend Store

  • MYTH_SESSION_STORAGE_DIR overrides the local path.
  • Default store: services/backend/.local/myth-sessions.
  • POST /v1/npc-ticks appends sorted bounded ticks.
  • GET /v1/myth-sessions/{session_id}/history returns session plus ticks.

Safety Boundary

  • No raw photos, scan binaries, or provider data URIs in history JSON.
  • No Meshy/OpenAI keys, bearer tokens, or raw provider payload markers.
  • No raw personal email, chat, calendar, or document bodies.
  • Not production account memory or autonomous background NPC memory.
