# P0.35 3D Provider Evaluation Suite Regression

Date: 2026-06-06

## Scope

P0.35 adds a canonical `default-v0` 3D provider evaluation suite with 20
non-private prompt cases. The suite is run through `evaluate-3d` and records
case categories, generated asset provenance, variant coverage, SceneKit-loadable
coverage, elapsed time, sanitized errors, and manual review scoring fields.

The suite is local/no-key by default. Running it through Meshy remains explicit
and requires backend-only `MESHY_API_KEY`.
