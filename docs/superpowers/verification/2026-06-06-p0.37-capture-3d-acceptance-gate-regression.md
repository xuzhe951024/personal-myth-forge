Date: 2026-06-06

## Scope

P0.37 adds a final acceptance check named `capture_3d_acceptance`. The check
creates a synthetic local `guided_scan` capture, reads three stored source
images through the same helper used by `POST /v1/myth-sessions/from-capture`,
passes those source images into the 3D provider request, and reports safe
multi-image provenance.
