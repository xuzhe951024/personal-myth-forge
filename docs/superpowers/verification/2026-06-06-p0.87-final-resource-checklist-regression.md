# P0.87 Final Resource Checklist Regression

Date: 2026-06-06
Branch: `codex/p0.87-final-resource-checklist`

## Scope

P0.87 surfaces backend `final_resources_preflight.items` in the iPhone
`Final Launch Status` panel. Operators can now see which required resource,
provider key, or deploy slot is missing or blocked without reading backend JSON.

The checklist consumes only sanitized readiness metadata. It does not expose
provider secrets, raw personal context, local filesystem paths, or raw media.

No backend provider call, deploy command, signing config, Xcode license command,
or global system state changed.
