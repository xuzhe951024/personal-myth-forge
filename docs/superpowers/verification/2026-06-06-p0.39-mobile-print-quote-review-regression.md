# P0.39 Mobile Print Quote Review Regression

Date: 2026-06-06

## Scope

P0.39 connects the P0.38 backend print quote contract to the iOS app source. It
adds Swift mobile quote models, a `POST /v1/print-quotes` API client method, a
SwiftUI `PrintQuoteReviewView`, and root view state for quote loading/errors.

This iteration does not add checkout, payment, order submission, live
Treatstock/Sculpteo calls, or device deployment.
