# P0.110 Final Handoff Index Regression

## Scope

P0.110 adds a read-only final handoff index for the final operator path:

- `services/backend/src/myth_forge_api/final_handoff_index.py`
- `myth_forge_api.cli final-handoff-index`
- `make final-handoff-index`
- `services/backend/.local/final-handoff-index.json`

The report indexes local rehearsal artifacts, composes the configured preflight
packet, and lists deploy/live-acceptance commands. It does not run commands,
call providers, apply resources, start servers, run Xcode, mutate signing or
keychain state, or write backend/iOS local config.
