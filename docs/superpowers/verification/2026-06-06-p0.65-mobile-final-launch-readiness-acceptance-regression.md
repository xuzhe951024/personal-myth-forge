# P0.65 Mobile Final Launch Readiness Acceptance Regression

Date: 2026-06-06

## Scope

P0.65 adds `mobile_final_launch_readiness_acceptance` to quick and full final
acceptance. The gate verifies:

- `GET /v1/final-demo-launch?mode=local`
- invalid mode rejection
- Swift model/API/root-view/device-preflight/contract-test source handoff
- no-secret, no-raw-media, no-payment-link, no-absolute-path, no-live-call default safety

## Notes

The quick final acceptance exit code remains `2` because the local iOS deploy
config and Apple SDK license are external machine blockers. P0.65's new gate
passes and is not the blocker.

The final launch command examples now write to ignored `.local/*.json` paths
instead of `/tmp/*.json`, so the read-only endpoint can truthfully report no
absolute local paths.
