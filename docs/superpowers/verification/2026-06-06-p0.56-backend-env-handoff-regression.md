# P0.56 Backend Env Handoff Regression

Date: 2026-06-06

## Scope

P0.56 adds a project-local writer for the ignored backend provider env:

```bash
MESHY_API_KEY=... \
OPENAI_API_KEY=... \
make backend-write-provider-env
```

The writer creates `services/backend/.env`, selects `THREE_D_PROVIDER=meshy` and
`NPC_PROVIDER=openai`, redacts command output, and leaves live provider
validation to explicit handoff/acceptance commands.

## Safety Boundary

- The writer only writes ignored `services/backend/.env`.
- It refuses tracked `services/backend/.env`.
- It rejects missing required Meshy/OpenAI keys and multiline values.
- It does not print key values to stdout/stderr.
- It does not call Meshy, OpenAI, print providers, Xcode, keychain, Apple
  services, or global shell/profile mutation commands.
