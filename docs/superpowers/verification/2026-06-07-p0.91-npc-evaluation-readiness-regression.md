## Commands

Create or refresh the source readiness report with the no-key local NPC suite:

```bash
cd services/backend
uv run python -m myth_forge_api.cli evaluate-npc \
  --provider local \
  --suite default-v0 \
  --tick-steps 2 \
  --output .local/npc-evaluation-local.json
```
