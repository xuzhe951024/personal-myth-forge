# P0.68 iOS Local Network Readiness Regression

Date: 2026-06-06

## Scope

P0.68 adds source-verified local network privacy readiness for the physical
iPhone demo. The app now declares `NSLocalNetworkUsageDescription` in
`Info.plist`, and the existing source-only gates require it.

This closes a deploy-facing gap for the LAN backend handoff:

- backend runs on a device-reachable URL such as `http://192.168.1.10:8080`
- mobile reads `PMF_BACKEND_BASE_URL` from deploy config
- App Transport Security allows local networking
- iOS privacy purpose string is present before a physical-device demo
