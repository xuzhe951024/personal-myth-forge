# P0.81 Mobile Generation Provenance Regression

## Scope

P0.81 turns generated asset provenance into a tested iPhone summary. The Artifact
screen can now show the 3D generation route, selected image count, scan asset
count, provider route, provider selection reason, and raw-source privacy state.

This directly supports the final demo path where a phone capture or ARKit/Object
Capture package must visibly reach the backend 3D generation route.

## Safety

The new summary builder sanitizes provider route and selection reason text. It
does not expose raw media, `local-capture://` URIs, local file paths, provider
secrets, or payment links.
