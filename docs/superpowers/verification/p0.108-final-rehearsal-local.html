Local Final Rehearsal

project-root handoff reports for the iPhone final showcase

No-Key
Canonical command make final-rehearsal-local

Execution Order

Evaluate local 3D and NPC Agentmake backend-evaluate-local
Write local final acceptancemake final-acceptance-local accepts exit 0 or 2
Write final demo launch reportmake final-demo-launch
Write iOS deploy runbookmake ios-deploy-runbook-local accepts exit 0 or 2

Deploy runbookservices/backend/.local/ios-deploy-runbook-local.json

Safety Contract

Local rehearsal does not call live providers, apply provider secrets, start servers, run Xcode, accept Apple licenses, write keychain entries, sign apps, or mutate global machine state.

Exit 2 is allowed only when a usable blocked report was written for expected local iPhone deploy or Xcode gates. Exit 1 still fails the rehearsal.

Source gates: backend tests, iOS showcase acceptance, Swift project checks, and final acceptance all inspect this handoff path.
