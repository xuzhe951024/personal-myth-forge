# P0.40 Resource Handoff Bundle Regression

Date: 2026-06-06

## Scope

P0.40 adds a sanitized `resource-handoff` CLI/report for final operator
resource provisioning. It consolidates backend Meshy/OpenAI key requirements,
print provider key slots, iOS deployment values, the Apple SDK build gate, next
commands, and leak checks.

This iteration does not create or validate real provider keys, call Meshy,
OpenAI, Treatstock, Sculpteo, or Apple services, mutate `.env`, edit signing,
change global Xcode state, or accept Apple SDK licenses.
