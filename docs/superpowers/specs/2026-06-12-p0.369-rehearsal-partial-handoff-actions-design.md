# P0.369 Rehearsal Partial Handoff Actions Design

## Goal

Make the local rehearsal aggregation include high-priority handoff actions from `final-demo-launch-local` even when that report is `partial`, so the final iOS device rehearsal report actually exposes provider and Treatstock print handoff actions.

## Current Evidence

- `final-demo-launch-local.json` is `partial` and already contains the provider handoff and Treatstock configured quote handoff actions near the top.
- `ios-device-launch-rehearsal.json` embeds `final_demo_launch_local` in `final_rehearsal_local.reports`, but `final_rehearsal_local.operator_actions` skips it because `_local_rehearsal_operator_actions` only considers `missing` and `blocked` sources.
- P0.368 can prioritize provider/print actions only after they enter the rehearsal action list; this follow-up makes the local source collector include those partial handoff actions.

## Behavior

`_local_rehearsal_operator_actions(local_sources)` should:

- Continue handling missing and blocked source reports as before.
- Also inspect partial source reports with nested `operator_actions`.
- From partial source reports, include only provider and print handoff actions. Do not copy generic final resource or optional actions.
- Preserve the six-action cap, source-prefix normalization, validation-command retention, and existing mobile deploy detail behavior.

## Non-Goals

- Do not treat partial reports as ready.
- Do not run live providers, Xcode, signing, or print quote/order actions.
- Do not include every partial report action; only handoff actions that unblock the final 3D/NPC provider and print lanes.

## Acceptance

- A focused RED/GREEN test proves provider and print handoff actions from partial `final_demo_launch_local` survive local rehearsal aggregation.
- Existing iOS device launch rehearsal and adjacent handoff tests remain green.
- Refreshed `ios-device-launch-rehearsal.json` shows provider and Treatstock handoff actions near the top while remaining blocked on external device/provider/print evidence.
