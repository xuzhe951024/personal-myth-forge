# P0.217 Final Resource Next Action Design

## Context

The final demo launch report is currently blocked first by `apply_final_resources`, which is in turn blocked by `final_resource_requirements:MESHY_API_KEY`. The resource requirements report already has `first_blocker`, `operator_actions`, and validation commands. The iPhone panel renders those rows, but it does not expose a stable structured "next input" card for the operator who will provide final Meshy/OpenAI/iOS resources at the end.

## Goal

Add `next_action` to `final_resource_requirements_report` and surface it on the iPhone Final Launch resource requirements panel.

## Non-Goals

- Do not write `.env`, `Deployment.local.xcconfig`, or `final-resources.env`.
- Do not call Meshy, OpenAI, Treatstock, Xcode, signing, keychain, or live providers.
- Do not change which resources are required or ready.

## Backend Contract

For blocked reports, `next_action` is derived from `first_blocker`:

```json
{
  "id": "MESHY_API_KEY",
  "label": "Meshy API key",
  "status": "missing",
  "classification": "missing_required_value",
  "command": "provide MESHY_API_KEY in final-resources.env",
  "detail": "Backend-only secret for live Meshy 3D generation.",
  "domain": "backend_provider",
  "destination": "services/backend/.local/final-resources.env",
  "validation_command": "make final-resources-preflight",
  "source": "first_blocker"
}
```

Ready reports set `next_action` to `null`.

## Mobile Contract

Swift decodes `next_action` as `FinalResourceRequirementsNextAction`. `FinalLaunchMobileSummary.resourceRequirementRows` shows:

```text
Next input: MESHY_API_KEY missing | provide MESHY_API_KEY in final-resources.env | services/backend/.local/final-resources.env | make final-resources-preflight
```

The row is sanitized using the same final launch redaction path.

## Visual And Source Gates

The source gate requires `next_action`, `FinalResourceRequirementsNextAction`, and `Next input:`. Visual regression adds one static iPhone artifact, increasing the checked-in artifact count from 24 to 25.
