# P0.265 External Ledger Destination Actions Design

## Goal

Make `final-external-action-ledger` operator actions tell the operator where to provide missing final resources and which command to rerun afterward.

## Context

The external action ledger is the Mac-side checklist for things Codex must not do automatically: secrets, account resources, Xcode/global actions, live provider cost actions, and physical device runtime work.

Its resource action records already include:

- `destination`, such as `services/backend/.local/final-resources.env`
- `command`, such as `make final-resources-preflight`

But top-level `operator_actions` currently say only `provide MESHY_API_KEY`. That is less actionable than the other final handoff reports, which already use wording like `provide MESHY_API_KEY in final-resources.env`.

## Design

Update only the top-level operator action text for missing resource input actions:

- Format missing resource inputs as `provide <slot> in final-resources.env; rerun make final-resources-preflight`.
- Use the basename of `destination` instead of the full repo-local path, avoiding local path leakage.
- Preserve existing behavior for blocked actions, live cost consent actions, manual global actions, de-duplication, statuses, source reports, and safety flags.
- Do not write resource files, call providers, launch Xcode, alter Keychain, or mutate global state.

## Test Plan

Extend `test_external_action_ledger_blocks_missing_resources_without_running_actions()` to assert the first two top-level operator actions include the destination and validation command:

- `provide MESHY_API_KEY in final-resources.env; rerun make final-resources-preflight`
- `provide OPENAI_API_KEY in final-resources.env; rerun make final-resources-preflight`

The test should fail before implementation because the old action text omits destination and validation.

## Self-Review

- No placeholders.
- Scope is one formatting improvement in an existing report.
- The change improves external resource handoff while preserving the no-secret/no-side-effect contract.
