# P0.214 Showcase Evidence Visual Regression Design

## Goal

Add a checked-in 390x844 iPhone visual artifact for the P0.213 Showcase Evidence panel so the visual regression index proves the consolidated capture, generation, SceneKit, NPC Agent, and print handoff evidence remains visible.

## Context

P0.213 added the in-app `ShowcaseEvidenceView` and summary builder. The existing visual regression gate still counts 21 artifacts and does not include a static proof for that new panel, so the source contract can pass while the new evidence surface has no visual snapshot.

## Scope

This iteration only changes the visual evidence chain:

- Add one `VisualArtifactSpec` named `p0.214_showcase_evidence_visual`.
- Add a static HTML proof and 390x844 PNG screenshot.
- Add a notes file describing the viewport, evidence represented, and privacy constraints.
- Update tests, README, and iOS showcase acceptance source requirements from 21 artifacts to 22.

No Swift runtime code, provider calls, Xcode commands, live 3D generation, print quote calls, or global machine state changes are part of this iteration.

## Required Visual Tokens

The artifact must contain:

- `Showcase Evidence`
- `Showcase evidence ready`
- `Capture-to-3D`
- `Generation Result`
- `SceneKit Load`
- `NPC Agent`
- `Print Handoff`
- `3 saved ticks`
- `Local file paths and payment links are withheld.`

The HTML and notes must avoid the existing visual safety forbidden patterns, including local paths, raw capture URIs, provider keys, raw payment links, and raw image data.

## Testing

Use TDD:

1. Update the expected visual artifact list and summary count to 22, then run the focused tests and observe RED because the implementation is missing.
2. Add the default artifact spec, static HTML, notes, screenshot, README text, and acceptance source requirements.
3. Run the focused visual regression tests, `make visual-regression`, iOS showcase acceptance tests touched by the count update, backend lint, and `git diff --check`.

## Self-Review

- The design covers the missing P0.213 visual evidence.
- The artifact is read-only and checked in.

- The privacy text is descriptive and does not include forbidden raw identifiers.
