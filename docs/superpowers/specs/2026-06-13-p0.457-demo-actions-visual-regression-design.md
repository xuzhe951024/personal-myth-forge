# P0.457 Demo Actions Visual Regression Design

## Context

P0.455 made the iPhone Demo Script and Showcase Autopilot stop on the external action ledger before final launch. P0.456 added backend mobile final launch readiness acceptance coverage for that source path. The default visual regression set still lacks a mobile-width artifact showing the new Demo Script `External Actions` row and `Check Actions` autopilot state.

## Goal

Add a default visual regression artifact that proves the guided iPhone demo visibly surfaces external action blockers, including `backend-device-demo`, before final launch.

## Scope

- Add a new `VisualArtifactSpec` with id `p0.457_demo_script_external_actions`.
- Add a static 390x844 mobile HTML proof showing:
  - `Demo Script`
  - `External Actions`
  - `Check Actions`
  - `backend-device-demo`
  - `Review external action blockers`
  - `Final Launch`
  - local-only safety copy
- Add a matching PNG screenshot and regression notes.
- Update visual regression tests to include the new artifact and expected summary count.

## Non-Goals

- No live provider calls, Xcode signing, iPhone execution, or `.local` report refresh.
- No Swift runtime behavior changes.
- No payment, Treatstock quote, or external order action.
